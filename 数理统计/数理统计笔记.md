# 第一章、参数估计

对数似然函数：$l(θ)=∑_{i=1}^{n} \ln f(X_i;θ)$

Fisher 信息量（样本）：$I_n(θ)=−E[\frac{d^2l(θ)}{dθ^2}]$

单样本 Fisher 信息量：$I1(θ)=−E[\frac{d^2\ln f(X;θ)}{dθ^2}]$，且 $I_n(θ)=nI_1(θ)$

单样本对数似然函数->二阶导数->代入EX（如果二阶导表达式中还有x）

|        对比维度        |               **完全充分统计量的因子分解定理**               |        **UMPT 相关的因式分解逻辑（含 N-P 引理核心）**        |
| :--------------------: | :----------------------------------------------------------: | :----------------------------------------------------------: |
|      **核心目标**      | 判定一个统计量 T(X) 是否为 **充分统计量**（压缩样本信息而不损失参数信息），再结合完全性定义判定**完全充分统计量** | 构造**最优检验**：在简单假设下构造 MPT，在复合假设下构造 UMPT，核心是找到最优拒绝域的形式 |
|      **适用场景**      | 任意参数模型（离散 / 连续型分布），针对**参数 \**θ\**** 的统计量判定 | 1. 简单假设：H0:θ=θ0 vs H1:θ=θ1<br />2. 复合假设：H0:θ∈Θ0 vs H1:θ∈Θ1（需结合完全充分统计量 + 无偏性） |
|    **核心定理表述**    | **因子分解定理（充分性判定）**设总体的密度 / 质量函数为 f(x;θ)，X=(X1,…,Xn) 为样本，则 T(X) 是 θ 的**充分统计量**的充要条件是：样本的联合密度 / 质量函数可分解为L(θ;X)=g(T(X);θ)⋅h(X)其中 g 是关于 T 和 θ 的非负函数，h 是仅关于样本 X 而**与 \**θ\** 无关**的非负函数。<br />**完全性补充**：若充分统计量 T 满足 “对任意可测函数 g，Eθ[g(T)]=0,∀θ∈Θ⟹g(T)=0,a.s.”，则 T 为完全充分统计量。 | **1. 简单假设（N-P 引理核心因式分解）**MPT 的拒绝域由**似然比**的因式分解决定：$λ(X)=\frac{L(θ_0;X)}{L(θ_1;X)}=\frac{g_0(T;θ_0)⋅h(X)}{g_1(T;θ_1)⋅h(X)}=\frac{g_0(T;θ_0)}{g_1(T;θ_1)}$即似然比可分解为**仅关于充分统计量 T 和参数的函数**，因此 MPT 的拒绝域等价于 T 的某个截断区域。<br />**2. 复合假设（UMPT 构造）**若存在完全充分统计量 T，则 UMPT 的拒绝域可由 T 的函数构造（结合无偏性：Eθ[ϕ(T)]=α,∀θ∈Θ0）。 |
| **因式分解的核心作用** | 分离**含参数的信息**（集中在 g(T;θ)）和**不含参数的冗余信息**（集中在 h(X)），从而判定 T 是否能 “浓缩” 样本中关于 θ 的全部信息。 | 消去样本的冗余信息（h(X) 被约去），证明**似然比仅依赖充分统计量 T**，因此最优拒绝域无需使用全部样本，仅需使用 T 即可，简化 UMPT 的构造。 |
|      **关键对象**      |      统计量 T(X)、样本联合密度 L(θ;X)、分解函数 g 和 h       |  似然比 λ(X)、检验函数 ϕ(X)、充分统计量 T(X)、显著性水平 α   |
|   **与完全性的关系**   | 完全性是**独立于充分性的额外性质**，因子分解定理仅判定充分性，需结合完全性定义才能得到完全充分统计量 | 完全充分统计量是构造**复合假设下 UMPT**的核心工具（如单边检验的 UMPT 存在性证明）；简单假设下的 MPT 无需完全性，仅需充分性。 |
| **离散 / 连续型差异**  | 无本质差异，因式分解形式完全相同，仅密度 / 质量函数的符号不同 | 连续型：似然比连续，可直接取阈值 k 控制 α；<br />离散型：似然比离散，需引入随机化概率 r 填补概率断层，才能精确控制 α。 |
|      **核心结论**      | 满足因式分解的 T 是充分统计量，再满足完全性即为完全充分统计量 ——**完全充分统计量是参数的最优信息载体** | 1. 简单假设：似然比的因式分解证明 MPT 的拒绝域等价于 T 的截断区域，N-P 引理保证其最优性；2. 复合假设：完全充分统计量保证 UMPT 的存在性，其拒绝域由 T 的函数唯一确定。 |
