$$\frac{c - \mu_0}{\sqrt{\frac{\sigma_0^2}{n}}} = u_{1-\alpha}$$

$$c = \mu_0 + \frac{\sigma_0}{\sqrt{n}} u_{1-\alpha}$$

$$\varphi(x) = \begin{cases} 1, & \bar{x} \ge \mu_0 + \frac{\sigma_0}{\sqrt{n}} u_{1-\alpha} \\ 0, & \bar{x} < \mu_0 + \frac{\sigma_0}{\sqrt{n}} u_{1-\alpha} \end{cases} = \begin{cases} 1, & \frac{\bar{x} - \mu_0}{\sqrt{\frac{\sigma_0^2}{n}}} \ge u_{1-\alpha} \\ 0, & \frac{\bar{x} - \mu_0}{\sqrt{\frac{\sigma_0^2}{n}}} < u_{1-\alpha} \end{cases}$$

势函数定义为在参数为 $\mu$ 时拒绝原假设的概率：

$$\begin{aligned} E_{\mu}(\varphi(x)) &= P_{\mu} \left\{ \frac{\bar{x} - \mu_0}{\sqrt{\frac{\sigma_0^2}{n}}} \ge u_{1-\alpha} \right\} \\ &= P_{\mu} \left\{ \frac{\bar{x} - \mu}{\sqrt{\frac{\sigma_0^2}{n}}} \ge u_{1-\alpha} + \frac{\mu_0 - \mu}{\sqrt{\frac{\sigma_0^2}{n}}} \right\} \\ &= 1 - \Phi \left( u_{1-\alpha} + \frac{\mu_0 - \mu}{\sqrt{\frac{\sigma_0^2}{n}}} \right) \end{aligned}$$

$$E_{\mu}(\varphi(x)) = 1 - \Phi \left( u_{1-\alpha} + \frac{\mu_0 - \mu}{\sqrt{\frac{\sigma_0^2}{n}}} \right), \quad \mu \le \mu_0$$

犯第二类错误的概率（取伪）：

当 $\mu > \mu_0$ 时，备择假设 $H_1$ 为真，此时犯第二类错误的概率 $\beta$ 为：



$$\beta = 1 - E_{\mu}(\varphi(x)) = \Phi \left( u_{1-\alpha} + \frac{\mu_0 - \mu}{\sqrt{\frac{\sigma_0^2}{n}}} \right), \quad \mu > \mu_0$$

第二类错误单调性

无偏检验

在备择假设 $H_1$ 区域内的拒绝概率始终不小于其在原假设 $H_0$ 区域内的拒绝概率（即不小于显著性水平 $\alpha$），则称该检验为**无偏检验**。

#### (1) 验证过程

- 显著性水平： 在边界点 $\mu = \mu_0$ 处，势函数的值恰好为：

  

  $$g(\mu_0) = 1 - \Phi(u_{1-\alpha} + 0) = 1 - (1 - \alpha) = \alpha$$

- **原假设区域 ($\mu \le \mu_0$)：** 由于 $g(\mu)$ 单调递增，所以在该区域内有 $g(\mu) \le g(\mu_0) = \alpha$。这符合控制第一类错误的要求。

- 备择假设区域 ($\mu > \mu_0$)： 同样由于 $g(\mu)$ 单调递增，在该区域内必有：

  

  $$g(\mu) > g(\mu_0) = \alpha$$

#### (2) 结论

因为在备择假设 $H_1$ 下，拒绝原假设的概率（功效）始终大于显著性水平 $\alpha$，所以：

该 UMPT 检验是一个无偏检验。

### （1）求 $\sigma^2$ 的极大似然估计 $\hat{\sigma}^2$

写出似然函数：

由于 $\mu_0$ 已知，总体密度函数为 $f(x; \sigma^2) = \frac{1}{\sqrt{2\pi(2\sigma^2)}} \exp\left( -\frac{(x-\mu_0)^2}{2(2\sigma^2)} \right)$。

样本的似然函数为：



$$L(\sigma^2) = \prod_{i=1}^n \frac{1}{2\sqrt{\pi\sigma^2}} \exp\left( -\frac{(x_i-\mu_0)^2}{4\sigma^2} \right) = (4\pi\sigma^2)^{-\frac{n}{2}} \exp\left( -\frac{\sum_{i=1}^n (x_i-\mu_0)^2}{4\sigma^2} \right)$$

取对数：



$$\ln L(\sigma^2) = -\frac{n}{2}\ln(4\pi) - \frac{n}{2}\ln(\sigma^2) - \frac{\sum_{i=1}^n (x_i-\mu_0)^2}{4\sigma^2}$$

对 $\sigma^2$ 求导并令其为 0：



$$\frac{d \ln L}{d(\sigma^2)} = -\frac{n}{2\sigma^2} + \frac{\sum_{i=1}^n (x_i-\mu_0)^2}{4(\sigma^2)^2} = 0$$



解得 $\sigma^2$ 的极大似然估计量为：



$$\hat{\sigma}^2 = \frac{1}{2n} \sum_{i=1}^n (x_i - \mu_0)^2$$

### (2)求 $\sigma^2$ 的一致最小方差无偏估计 (UMVUE)

验证无偏性：

已知 $\frac{(x_i-\mu_0)}{\sqrt{2\sigma^2}} \sim N(0,1)$，则 $\frac{(x_i-\mu_0)^2}{2\sigma^2} \sim \chi^2(1)$。

由此可得 $E[(x_i-\mu_0)^2] = 2\sigma^2$。

对于极大似然估计量 $\hat{\sigma}^2$：



$$E[\hat{\sigma}^2] = \frac{1}{2n} \sum_{i=1}^n E[(x_i-\mu_0)^2] = \frac{1}{2n} \cdot n(2\sigma^2) = \sigma^2$$



因此，$\hat{\sigma}^2$ 已经是 $\sigma^2$ 的无偏估计。

寻找充分完全统计量：

该分布属于指数分布族，其充分完全统计量为 $T = \sum_{i=1}^n (x_i - \mu_0)^2$。

结论：

根据 Lehmann-Scheffé 定理，由于 $\hat{\sigma}^2$ 是充分完全统计量 $T$ 的函数且是无偏的，所以：

$\sigma^2$ 的 UMVUE 为 $\hat{\sigma}^2 = \frac{1}{2n} \sum_{i=1}^n (x_i - \mu_0)^2$。

### (3)问该 UMVUE 是否为有效估计？并证明。

**结论：是有效估计。**

**证明方法：计算其方差并与 Cramér-Rao (C-R) 下界对比。**

1. 计算估计量的方差：

   已知 $\frac{\sum(x_i-\mu_0)^2}{2\sigma^2} \sim \chi^2(n)$，卡方分布的方差为自由度的 2 倍。

   

   $$Var\left( \frac{\sum(x_i-\mu_0)^2}{2\sigma^2} \right) = 2n \Rightarrow \frac{1}{4\sigma^4} Var\left( \sum(x_i-\mu_0)^2 \right) = 2n$$

   $$Var(\hat{\sigma}^2) = Var\left( \frac{1}{2n} \sum(x_i-\mu_0)^2 \right) = \frac{1}{4n^2} \cdot 8n\sigma^4 = \frac{2\sigma^4}{n}$$

2. 计算 C-R 下界：

   对数似然函数的二阶导数为 $\frac{d^2 \ln L}{d(\sigma^2)^2} = \frac{n}{2(\sigma^2)^2} - \frac{\sum(x_i-\mu_0)^2}{2(\sigma^2)^3}$。

   Fisher 信息量 $I_n(\sigma^2) = -E\left[ \frac{d^2 \ln L}{d(\sigma^2)^2} \right] = -\frac{n}{2\sigma^4} + \frac{n(2\sigma^2)}{2\sigma^6} = \frac{n}{2\sigma^4}$。

   C-R 下界为 $\frac{1}{I_n(\sigma^2)} = \frac{2\sigma^4}{n}$。

**结论证明：** 由于 $Var(\hat{\sigma}^2) = \frac{2\sigma^4}{n}$ 恰好等于 C-R 下界，故该估计量是有效估计。

## 三、设 $x_1, x_2, \cdots, x_n$ 是来自正态总体 $N(\mu, \sigma_0^2)$ 的简单样本，其中 $\sigma_0^2$ 为已知常数，$\mu$ 为未知参数。考虑假设检验问题：$$H_0: \mu = \mu_0 \quad \text{vs} \quad H_1: \mu < \mu_0$$

### (1) 求显著性水平 $\alpha$ 的一致最大优势检验 (UMPT)

1. 确定检验统计量：

   在正态总体均值检验中，样本均值 $\bar{x} = \frac{1}{n}\sum_{i=1}^n x_i$ 是 $\mu$ 的充分统计量。由于备择假设为 $\mu < \mu_0$，这是一个左侧检验，拒绝域的形式应为 $\bar{x} < c$。

2. 利用显著性水平确定临界值：

   根据显著性水平 $\alpha$ 的定义，当 $H_0$ 为真（即 $\mu = \mu_0$）时，拒绝 $H_0$ 的概率应为 $\alpha$：

   

   $$P_{\mu_0}(\bar{x} < c) = \alpha$$

   

   由于 $\bar{x} \sim N(\mu_0, \frac{\sigma_0^2}{n})$，将其标准化：

   

   $$P_{\mu_0}\left( \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}} < \frac{c - \mu_0}{\sigma_0 / \sqrt{n}} \right) = \Phi\left( \frac{c - \mu_0}{\sigma_0 / \sqrt{n}} \right) = \alpha$$

   

   由标准正态分布分位数定义 $\Phi(u_\alpha) = \alpha$（或写为 $-u_{1-\alpha}$），可得：

   

   $$\frac{c - \mu_0}{\sigma_0 / \sqrt{n}} = u_\alpha$$

   

   解得临界值 $c = \mu_0 + u_\alpha \frac{\sigma_0}{\sqrt{n}}$。

3. 给出 UMPT 检验函数：

   

   $$\varphi(x) = \begin{cases} 1, & \bar{x} < \mu_0 + u_\alpha \frac{\sigma_0}{\sqrt{n}} \\ 0, & \bar{x} \ge \mu_0 + u_\alpha \frac{\sigma_0}{\sqrt{n}} \end{cases}$$

   

   或者表示为标准化形式：当 $\frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}} < u_\alpha$ 时，拒绝 $H_0$。

------

### (2) 求 UMPT 的势函数及第二类错误概率 $\beta(\mu)$

1. 势函数 $g(\mu)$：

   势函数定义为拒绝原假设的概率。

   

   $$g(\mu) = E_\mu[\varphi(x)] = P_\mu\left( \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}} < u_\alpha \right)$$

   

   对左侧项进行标准化处理：

   

   $$g(\mu) = P_\mu\left( \frac{\bar{x} - \mu}{\sigma_0 / \sqrt{n}} < u_\alpha + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} \right) = \Phi\left( u_\alpha + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} \right)$$

2. 第二类错误概率 $\beta(\mu)$：

   当 $H_1$ 为真（即 $\mu < \mu_0$）时，犯第二类错误（取伪）的概率为：

   

   $$\beta(\mu) = 1 - g(\mu) = 1 - \Phi\left( u_\alpha + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} \right)$$

------

### (3) 讨论 $\beta(\mu)$ 关于 $\mu$ 的单调性

我们分析 $\beta(\mu)$ 随 $\mu$ 增加时的变化情况：

- **内部项分析：** 当真实的 $\mu$ **增加**时，分式项 $\frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}}$ 的分子 $(\mu_0 - \mu)$ 会**减小**。
- **复合单调性：**
  1. 随着 $\mu \uparrow$，括号内的整体数值 $z = u_\alpha + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}}$ 随之 **减小**。
  2. 由于标准正态累积分布函数 $\Phi(z)$ 是单调增加的，因此 $\Phi(z)$ 随之 **减小**。
  3. 最后，因为 $\beta(\mu) = 1 - \Phi(z)$，所以随着 $\Phi(z)$ 的减小，$\beta(\mu)$ 反而 **增加**。

**题目：** 设 $x_1, x_2, \cdots, x_n$ 来自正态总体 $N(0, \sigma^2)$ 的简单样本，对于给定的 $\alpha (0 < \alpha < \frac{1}{2})$，用 $z_{1-\alpha}$ 表示标准正态分布的 $1-\alpha$ 分位数，则 $P\{ \bar{x} \le \frac{\sigma}{\sqrt{n}} z_{1-\alpha} \} = \_\_\_\_$。

**解析：**

1. **确定样本均值分布：** 由于 $x_i \sim N(0, \sigma^2)$，则样本均值 $\bar{x} \sim N(0, \frac{\sigma^2}{n})$。

2. **标准化：** 将不等式左边的 $\bar{x}$ 标准化为标准正态随机变量 $Z = \frac{\bar{x} - 0}{\sigma/\sqrt{n}}$。

3. 计算：

   

   $$P\{ \bar{x} \le \frac{\sigma}{\sqrt{n}} z_{1-\alpha} \} = P\{ \frac{\bar{x}}{\sigma/\sqrt{n}} \le z_{1-\alpha} \} = \Phi(z_{1-\alpha})$$

4. **结论：** 根据分位数的定义 $\Phi(z_{1-\alpha}) = 1-\alpha$。

**答案：$1-\alpha$**

### 2. 构造卡方分布统计量

**题目：** 设 $x_1, x_2, \cdots, x_n$ 来自正态总体 $N(0, \sigma^2)$ 的简单样本，则当 $c = \_\_\_\_$ 时，随机变量 $c(\bar{x}^2 + \hat{\sigma}^2)$ 服从 $\chi^2$ 分布，其中 $\bar{x} = \frac{1}{n}\sum x_i$，$\hat{\sigma}^2 = \frac{1}{n}\sum (x_i - \bar{x})^2$。

**解析：**

1. **化简统计量：** 注意到 $\bar{x}^2 + \hat{\sigma}^2 = \bar{x}^2 + \frac{1}{n}\sum x_i^2 - \bar{x}^2 = \frac{1}{n}\sum x_i^2$。
2. **构造卡方分布：** 已知 $\frac{x_i}{\sigma} \sim N(0, 1)$，由卡方分布定义，独立标准正态变量的平方和服从卡方分布，即 $\sum_{i=1}^n (\frac{x_i}{\sigma})^2 = \frac{1}{\sigma^2} \sum x_i^2 \sim \chi^2(n)$。
3. **确定 $c$：** 我们需要 $c \cdot (\frac{1}{n} \sum x_i^2) = \frac{1}{\sigma^2} \sum x_i^2$，从而解得 $c = \frac{n}{\sigma^2}$。

**答案：$\frac{n}{\sigma^2}$**（注：此时服从 $\chi^2(n)$ 分布）

## 二、

### 题目背景

设总体 $X$ 的密度函数为：



$$f(x; \theta) = \begin{cases} \frac{1}{\theta^2} x e^{-x/\theta}, & x > 0 \\ 0, & x \le 0 \end{cases}$$



其中 $\theta > 0$。$x_1, x_2, \cdots, x_n$ 是来自总体 $X$ 的简单样本。

------

### (1) 求 $\theta$ 的一致最小方差无偏估计 (UMVUE)

1. 寻找充分统计量：

写出样本的似然函数：



$$L(\theta) = \prod_{i=1}^n \frac{1}{\theta^2} x_i e^{-x_i/\theta} = \frac{1}{\theta^{2n}} \left( \prod_{i=1}^n x_i \right) e^{-\frac{1}{\theta} \sum_{i=1}^n x_i}$$



根据因子分解定理，充分统计量为 $T = \sum_{i=1}^n x_i$（或样本均值 $\bar{x}$）。由于该分布属于指数分布族，$T$ 也是完全统计量。

2. 构造无偏估计：

已知总体期望 $E(X)$ 为：



$$E(X) = \int_0^\infty x \cdot \frac{1}{\theta^2} x e^{-x/\theta} dx = \frac{1}{\theta^2} \int_0^\infty x^2 e^{-x/\theta} dx = \frac{1}{\theta^2} \cdot 2\theta^3 = 2\theta$$



因此，样本均值 $\bar{x}$ 的期望为 $E(\bar{x}) = E(X) = 2\theta$。

为了得到 $\theta$ 的无偏估计，令 $\hat{\theta} = \frac{\bar{x}}{2}$。其期望 $E(\frac{\bar{x}}{2}) = \frac{2\theta}{2} = \theta$。

3. 结论：

根据 Lehmann-Scheffé 定理，由于 $\frac{\bar{x}}{2}$ 是完全充分统计量 $\bar{x}$ 的函数且是无偏的，因此：

$\theta$ 的一致最小方差无偏估计 (UMVUE) 为 $\hat{\theta} = \frac{\bar{x}}{2}$。

------

### (2) 问 $\theta$ 的 UMVUE 是否为有效估计？说明理由。

**结论：是有效估计。**

理由证明：

判断一个无偏估计量是否为有效估计，需看其方差是否达到了 Cramér-Rao (C-R) 下界。

1. 计算估计量的方差：

已知 $Var(X) = E(X^2) - [E(X)]^2 = 6\theta^2 - (2\theta)^2 = 2\theta^2$。

则 $\hat{\theta} = \frac{\bar{x}}{2}$ 的方差为：



$$Var(\hat{\theta}) = Var\left( \frac{1}{2n} \sum_{i=1}^n x_i \right) = \frac{1}{4n^2} \sum_{i=1}^n Var(x_i) = \frac{1}{4n^2} \cdot n(2\theta^2) = \frac{\theta^2}{2n}$$

2. 计算 C-R 下界：

单个样本的对数似然函数为 $\ln f(x; \theta) = -2\ln\theta + \ln x - x/\theta$。

对 $\theta$ 求二阶导数：



$$\frac{\partial \ln f}{\partial \theta} = -\frac{2}{\theta} + \frac{x}{\theta^2}$$

$$\frac{\partial^2 \ln f}{\partial \theta^2} = \frac{2}{\theta^2} - \frac{2x}{\theta^3}$$



计算 Fisher 信息量 $I(\theta) = -E\left[ \frac{\partial^2 \ln f}{\partial \theta^2} \right] = -\frac{2}{\theta^2} + \frac{2(2\theta)}{\theta^3} = \frac{2}{\theta^2}$。

全体样本的 C-R 下界为：



$$\frac{1}{n I(\theta)} = \frac{1}{n(2/\theta^2)} = \frac{\theta^2}{2n}$$

结论：

由于 $Var(\hat{\theta}) = \frac{\theta^2}{2n}$ 恰好等于 C-R 下界，因此该估计量是有效估计。

这是最基础的方法，通过定义式结合 Gamma 函数的积分公式 $\int_0^\infty x^{n-1} e^{-ax} dx = \frac{\Gamma(n)}{a^n}$

### 题目背景

设 $x_1, x_2, \cdots, x_n$ 是来自正态总体 $N(0, \sigma^2)$ 的简单样本。考虑假设检验问题：



$$H_0: \sigma^2 = \sigma_0^2 \quad \text{vs} \quad H_1: \sigma^2 > \sigma_0^2$$

------

### (1) 求显著性水平 $\alpha$ 的似然比检验，并用检验函数 $\varphi(x)$ 表示

1. 写出似然函数：

由于均值已知为 0，似然函数为：



$$L(\sigma^2) = \prod_{i=1}^n \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left( -\frac{x_i^2}{2\sigma^2} \right) = (2\pi\sigma^2)^{-n/2} \exp\left( -\frac{\sum x_i^2}{2\sigma^2} \right)$$

2. 构造似然比统计量：

根据单调似然比（MLR）性质，对于正态方差的右侧检验，似然比是充分统计量 $T = \sum x_i^2$ 的增函数。因此，拒绝域的形式为：



$$W = \{ \mathbf{x} : \sum_{i=1}^n x_i^2 > c \}$$

3. 确定临界值 $c$：

在 $H_0$ 成立时（即 $\sigma^2 = \sigma_0^2$），已知 $\frac{\sum x_i^2}{\sigma_0^2} \sim \chi^2(n)$。

由显著性水平 $\alpha$ 的定义：



$$P_{\sigma_0^2} \{ \sum x_i^2 > c \} = P_{\sigma_0^2} \{ \frac{\sum x_i^2}{\sigma_0^2} > \frac{c}{\sigma_0^2} \} = \alpha$$



根据卡方分布的分位数定义，有 $\frac{c}{\sigma_0^2} = \chi_\alpha^2(n)$，即 $c = \sigma_0^2 \chi_\alpha^2(n)$。

4. 检验函数 $\varphi(x)$：



$$\varphi(x) = \begin{cases} 1, & \sum_{i=1}^n x_i^2 > \sigma_0^2 \chi_\alpha^2(n) \\ 0, & \sum_{i=1}^n x_i^2 \le \sigma_0^2 \chi_\alpha^2(n) \end{cases}$$

------

### (2) 求 $\varphi(x)$ 的势，并讨论其单调性

1. 求势函数 $g(\sigma^2)$：

势函数定义为在参数为 $\sigma^2$ 时拒绝原假设的概率：



$$g(\sigma^2) = E_{\sigma^2}[\varphi(x)] = P_{\sigma^2} \{ \sum x_i^2 > \sigma_0^2 \chi_\alpha^2(n) \}$$



标准化处理，利用 $\frac{\sum x_i^2}{\sigma^2} \sim \chi^2(n)$：



$$g(\sigma^2) = P_{\sigma^2} \{ \frac{\sum x_i^2}{\sigma^2} > \frac{\sigma_0^2}{\sigma^2} \chi_\alpha^2(n) \} = 1 - F_{\chi^2(n)} \left( \frac{\sigma_0^2}{\sigma^2} \chi_\alpha^2(n) \right)$$



其中 $F_{\chi^2(n)}$ 是自由度为 $n$ 的卡方分布累积分布函数。

**2. 讨论单调性：**

- **变量关系：** 当真实的方差 $\sigma^2$ **增大**时，分式 $\frac{\sigma_0^2}{\sigma^2}$ 会随之 **减小**。
- **函数性质：**
  1. 卡方分布的累积分布函数 $F_{\chi^2(n)}(z)$ 是关于 $z$ 的单调递增函数。
  2. 当 $z = \frac{\sigma_0^2}{\sigma^2} \chi_\alpha^2(n)$ 随 $\sigma^2$ 增大而减小时，$F_{\chi^2(n)}(z)$ 也随之 **减小**。
  3. 因此，$g(\sigma^2) = 1 - F_{\chi^2(n)}(z)$ 随之 **增大**。

结论：

势函数 $g(\sigma^2)$ 关于 $\sigma^2$ 是严格单调递增的。

这表明该检验是无偏的，即当 $\sigma^2 > \sigma_0^2$ 时，拒绝原假设的概率始终大于显著性水平 $\alpha$。

## 一、 填空题

### **1. $F$ 分布构造**

- **题目：** $X \sim N(\mu, \sigma^2)$，$c \cdot \frac{(x_1 - x_2)^2}{(x_3 - x_4)^2 + (x_5 - x_6)^2} \sim F(1, 2)$，求 $c$。
- **解析：**
  1. 构造 $\chi^2(1)$：$(x_1 - x_2) \sim N(0, 2\sigma^2)$，则 $\frac{(x_1 - x_2)^2}{2\sigma^2} \sim \chi^2(1)$。
  2. 构造 $\chi^2(2)$：同理，$\frac{(x_3 - x_4)^2}{2\sigma^2} + \frac{(x_5 - x_6)^2}{2\sigma^2} \sim \chi^2(2)$。
  3. 利用 $F$ 定义：$F = \frac{\chi^2(1)/1}{\chi^2(2)/2} = \frac{\frac{(x_1 - x_2)^2}{2\sigma^2}}{\frac{(x_3 - x_4)^2 + (x_5 - x_6)^2}{2\sigma^2} \cdot \frac{1}{2}} = \frac{2(x_1 - x_2)^2}{(x_3 - x_4)^2 + (x_5 - x_6)^2}$。
- **答案：** $c = 1$ （注意：若题目要求整体符合标准定义，$c$ 应对应分子分母自由度的配比，此处 $c=1$ 时统计量正好符合 $F(1,2)$ 的标准形式）。

### **2. 频率替换估计**

- **题目：** $X \sim N(1, \sigma^2)$，样本中有 $m$ 个小于 2，求 $\sigma$ 的频率替换估计 $\hat{\sigma}$。
- **解析：**
  1. 理论概率：$P(X < 2) = P(\frac{X - 1}{\sigma} < \frac{2 - 1}{\sigma}) = \Phi(\frac{1}{\sigma})$。
  2. 样本频率：$\hat{p} = \frac{m}{n}$。
  3. 令理论等于频率：$\Phi(\frac{1}{\sigma}) = \frac{m}{n} \Rightarrow \frac{1}{\sigma} = \Phi^{-1}(\frac{m}{n})$。
- **答案：** $\hat{\sigma} = \frac{1}{\Phi^{-1}(m/n)}$。

### **3. 无偏估计方差下界 (C-R 下界)**

- **题目：** $X_i \sim P(\lambda)$，求 $\lambda^2$ 无偏估计的方差下界。
- **解析：**
  1. 目标函数：$g(\lambda) = \lambda^2$，则 $g'(\lambda) = 2\lambda$。
  2. Fisher 信息量：泊松分布 $I(\lambda) = \frac{1}{\lambda}$，则 $I_n(\lambda) = \frac{n}{\lambda}$。
  3. C-R 下界公式：$\frac{[g'(\lambda)]^2}{I_n(\lambda)} = \frac{(2\lambda)^2}{n/\lambda} = \frac{4\lambda^3}{n}$。
- **答案：** $\frac{4\lambda^3}{n}$。

## 二、 极大似然与 UMVUE

- **题目：** $f(x; \theta) = \frac{1}{\theta^2} x e^{-x/\theta}, x > 0$（Gamma 分布 $\Gamma(2, \theta)$）。

- ### **(1) 极大似然估计：**

  1. $L(\theta) = \theta^{-2n} (\prod x_i) e^{-\sum x_i / \theta}$。
  2. $\ln L = -2n \ln \theta + \ln(\prod x_i) - \frac{\sum x_i}{\theta}$。
  3. $\frac{d \ln L}{d \theta} = -\frac{2n}{\theta} + \frac{\sum x_i}{\theta^2} = 0 \Rightarrow \hat{\theta} = \frac{\bar{x}}{2}$。

- ### (2) UMVUE：

  由于 $\hat{\theta} = \frac{\bar{x}}{2}$ 是充分完全统计量 $\bar{x}$ 的函数且 $E[\frac{\bar{x}}{2}] = \frac{2\theta}{2} = \theta$，根据莱曼-谢费定理，$\frac{\bar{x}}{2}$ 即为 $\theta$ 的 UMVUE。

## 三、 假设检验 (UMPT)

- **题目：** $X \sim N(\mu, \sigma^2)$，$\sigma^2$ 已知，$H_0: \mu \ge \mu_0$ vs $H_1: \mu < \mu_0$。

- ### (1) 求 UMPT：

  这是一个左侧检验。拒绝域为 $\bar{x} < \mu_0 - u_{1-\alpha} \frac{\sigma}{\sqrt{n}}$（或写为 $\bar{x} < \mu_0 + u_\alpha \frac{\sigma}{\sqrt{n}}$）。

- ### (2) 势是否大于 $\alpha$：

  是。 因为在备择假设区域 $H_1$ ($\mu < \mu_0$)，势函数 $g(\mu) = P_\mu(\text{拒绝} H_0)$ 随 $\mu$ 减小而增大。由于 $g(\mu_0) = \alpha$，当 $\mu < \mu_0$ 时，$g(\mu) > \alpha$。
