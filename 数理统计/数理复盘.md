# 判断题

## 1.相合、有效、无偏估计线性变换仍保持原有性质

对

1. 经过线性变换后“依旧保持”原有性质的估计

大多数核心的统计性质在线性变换下是极其稳健的：

- 无偏性 (Unbiasedness): 这是最直观的。如果 $E[\hat{\theta}] = \theta$，那么根据期望的线性性质：

  

  $$E[a\hat{\theta} + b] = aE[\hat{\theta}] + b = a\theta + b$$

  

  因此，无偏估计的线性变换依然是无偏的。

- **有效性 (Efficiency / Minimum Variance):** 如果 $\hat{\theta}$ 是方差最小的，那么 $Var(a\hat{\theta} + b) = a^2 Var(\hat{\theta})$。由于 $a^2$ 是常数因子，在所有线性变换后的竞争估计量中，$a\hat{\theta} + b$ 依然保持方差最小的地位（例如 **UMVUE** 在线性变换下依然是 UMVUE）。

- **相合性 (Consistency):** 根据概率极限的性质，如果 $\hat{\theta}_n \xrightarrow{P} \theta$，那么 $a\hat{\theta}_n + b \xrightarrow{P} a\theta + b$。这意味着大样本下的稳定性不会被线性变换破坏。

- **充分性 (Sufficiency):** 充分统计量关注的是样本中关于参数的信息提取。对参数进行一一对应的变换（线性变换显然是一一对应的，只要 $a \neq 0$），不会改变统计量所包含的信息量。

2. 经过线性变换后“不保持”或“有条件保持”的性质

这里需要特别注意，虽然性质本身可能“保持”，但其**形式**或**优良性**在非线性变换下会失效，而在**线性变换**下，唯一需要警惕的是那些与“原点”或“量纲”绝对位置相关的性质：

- **均方误差 (MSE) 的数值:** 虽然估计量的优劣排序通常不变，但 MSE 的绝对数值会随 $a^2$ 缩放。这不叫性质改变，但在做决策论分析时需要重新标定。
- **某些特定的区间估计 (Interval Estimation):** 虽然线性变换后的置信区间边界会同步移动，但如果变换涉及**负数**（即 $a < 0$），置信区间的上限和下限会发生翻转。如果不注意符号，结论会出错。

| **性质**             | **线性变换 (aθ+b)** | **非线性变换 (g(θ))**                                        |
| -------------------- | ------------------- | ------------------------------------------------------------ |
| **无偏性**           | **保持**            | **不保持** (因 $E[g(\hat{\theta})] \neq g(E[\hat{\theta}])$) |
| **极大似然性 (MLE)** | **保持**            | **保持** (MLE 具有不变性原理)                                |
| **相合性**           | **保持**            | **通常保持** (需 $g$ 连续)                                   |

## 2.一个参数的无偏估计的方差还达到了C-R不等式的下界，是该参数的UMVUE

对

1. 核心结论：充分性

如果一个无偏估计量 $\hat{\theta}$ 的方差达到了 **Cramér-Rao 下界 (CRLB)**，那么它不仅是 UMVUE（一致最小方差无偏估计），我们还给它一个专门的称号，叫做**有效估计量 (Efficient Estimator)**。

- **逻辑链条：** 达到 CRLB $\implies$ 它是所有无偏估计中方差最小的 $\implies$ 它是 UMVUE。

2. 为什么“达到下界”一定是“最小”？

C-R 不等式给出了在一定正则条件下，所有无偏估计量方差的“理论底线”：



$$Var(\hat{\theta}) \ge \frac{[g'(\theta)]^2}{nI(\theta)}$$

- 如果你的估计量已经踩到了这个“底线”，那么根据定义，没有任何其他无偏估计量的方差能比它更小了。
- 既然在每一点 $\theta$ 上它的方差都是最小的，它自然就符合 **Uniformly** (一致性) 的要求。

3. 容易混淆的点：反过来成立吗？（重点！）

这是考试和研究生面试中最爱考的陷阱：**UMVUE 的方差一定能达到 C-R 下界吗？**

**答案是：不一定。**

## 1.$x_1,x_2,……x_n$是服从泊松分布P(λ)的一组样本

### 答案：$\frac{\lambda}{n}$

在泊松分布 $P(\lambda)$ 中，我们通常使用样本均值 $\bar{X}$ 或样本总和 $\sum X_i$ 作为 $\lambda$ 的估计量。

------

### 1). 寻找充分统计量

对于泊松分布 $P(\lambda)$，其概率质量函数为 $P(X=x) = \frac{e^{-\lambda}\lambda^x}{x!}$。

根据因子分解定理，样本的联合概率为：



$$f(x_1, \dots, x_n; \lambda) = \prod_{i=1}^n \frac{e^{-\lambda}\lambda^{x_i}}{x_i!} = \frac{e^{-n\lambda}\lambda^{\sum x_i}}{\prod x_i!}$$



由此可见，$T = \sum_{i=1}^n X_i$ 是 $\lambda$ 的充分统计量。在实际估计中，我们通常取其线性变换 $\hat{\lambda} = \bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$，它也是 $\lambda$ 的唯一 UMVUE（一致最小方差无偏估计）。

------

### 2). 计算 $\bar{X}$ 的 MSE

在数理统计中，估计量 $\hat{\theta}$ 的均方误差定义为：



$$MSE(\hat{\theta}) = E[(\hat{\theta} - \theta)^2] = Var(\hat{\theta}) + [Bias(\hat{\theta})]^2$$

对于 $\hat{\lambda} = \bar{X}$：

1. 计算偏倚 (Bias)：

   由于 $X_i \sim P(\lambda)$，则 $E[X_i] = \lambda$。

   

   $$E[\bar{X}] = E\left[\frac{1}{n}\sum X_i\right] = \frac{1}{n} \cdot n\lambda = \lambda$$

   

   因为 $E[\bar{X}] = \lambda$，所以 $Bias(\bar{X}) = 0$（即 $\bar{X}$ 是无偏估计）。

2. 计算方差 (Variance)：

   由于 $X_i$ 是独立同分布样本，且 $Var(X_i) = \lambda$。

   

   $$Var(\bar{X}) = Var\left(\frac{1}{n}\sum X_i\right) = \frac{1}{n^2} \sum Var(X_i) = \frac{n\lambda}{n^2} = \frac{\lambda}{n}$$

3. 最终 MSE：

   

   $$MSE(\bar{X}) = \frac{\lambda}{n} + 0^2 = \frac{\lambda}{n}$$

------

### 3). 结论

对于参数 $\lambda$ 的充分统计量（以最常用的 $\bar{X}$ 为例），其均方误差（MSE）等于其方差：

> **$MSE(\bar{X}) = \frac{\lambda}{n}$**

## 2.$x_1,x_2,……x_n$是服从正态分布$N(μ,σ_0^2)$的一组样本，$μ^2$无偏估计且是完全充分统计量的结果？

### 答案

$\mu^2$ 的无偏估计且是完全充分统计量的结果是：

> **$\bar{X}^2 - \frac{\sigma^2}{n}$**

------

### 解析

1. 确定完全充分统计量：

   对于正态分布 $N(\mu, \sigma^2)$，当 $\sigma^2$ 已知时，指数族分布的形式决定了样本均值 $\bar{X}$ 是参数 $\mu$ 的完全充分统计量。

2. 构造无偏性：

   我们希望找到一个统计量 $g(\bar{X})$，使得 $E[g(\bar{X})] = \mu^2$。

   我们已知 $\bar{X} \sim N(\mu, \frac{\sigma^2}{n})$，利用方差计算公式：

   

   $$Var(\bar{X}) = E[\bar{X}^2] - (E[\bar{X}])^2$$

   

   将已知分布参数代入：

   

   $$\frac{\sigma^2}{n} = E[\bar{X}^2] - \mu^2$$

   

   通过简单的移项得：

   

   $$\mu^2 = E[\bar{X}^2] - \frac{\sigma^2}{n} = E\left[ \bar{X}^2 - \frac{\sigma^2}{n} \right]$$

3. 应用 Lehmann-Scheffé 定理：

   因为 $\bar{X}$ 是完全充分统计量，而 $g(\bar{X}) = \bar{X}^2 - \frac{\sigma^2}{n}$ 是 $\bar{X}$ 的函数且是 $\mu^2$ 的无偏估计，所以它就是唯一的一致最小方差无偏估计（UMVUE）。

## 3.设随机变量 $X \sim B(1, \theta)$，即 $P(X=1) = \theta, P(X=0) = 1-\theta$。现从该总体中抽取容量为 2 的样本，观测值为 $x_1 = 0, x_2 = 1$。已知参数 $\theta$ 的先验分布为离散分布，其可能取值为 $\{\frac{1}{4}, \frac{1}{2}, \frac{3}{4}\}$，且取每个值的先验概率均为 $P(\theta = \theta_i) = \frac{1}{3}$。 **求：** 在给定观测值 $(0, 1)$ 下，参数 $\theta$ 的后验期望 $E[\theta | x_1=0, x_2=1]$

### 答案：0.5

### 解析

第一步：计算似然函数 $L(\theta)$

由于样本 $x_1=0, x_2=1$ 是独立同分布抽样，似然函数为：



$$L(\theta) = P(X=0|\theta) \cdot P(X=1|\theta) = (1-\theta) \cdot \theta$$



针对 $\theta$ 的三个可能取值，分别计算似然值：

- 当 $\theta = 1/4$ 时，$L(1/4) = (1 - 1/4) \cdot 1/4 = 3/16$
- 当 $\theta = 1/2$ 时，$L(1/2) = (1 - 1/2) \cdot 1/2 = 1/4 = 4/16$
- 当 $\theta = 3/4$ 时，$L(3/4) = (1 - 3/4) \cdot 3/4 = 3/16$

第二步：计算全概率（边缘分布）

根据贝叶斯公式的分母，计算观测到 $(0, 1)$ 的总概率：



$$P(x) = \sum_{i=1}^3 P(\theta_i) L(\theta_i) = \frac{1}{3} \left( \frac{3}{16} + \frac{4}{16} + \frac{3}{16} \right) = \frac{1}{3} \cdot \frac{10}{16} = \frac{10}{48}$$

第三步：计算后验分布 $P(\theta | x)$

后验概率 $P(\theta_i | x) = \frac{P(\theta_i) L(\theta_i)}{P(x)}$：

- $P(\theta = 1/4 | x) = \frac{(1/3 \cdot 3/16)}{(10/48)} = \frac{3/48}{10/48} = \mathbf{0.3}$
- $P(\theta = 1/2 | x) = \frac{(1/3 \cdot 4/16)}{(10/48)} = \frac{4/48}{10/48} = \mathbf{0.4}$
- $P(\theta = 3/4 | x) = \frac{(1/3 \cdot 3/16)}{(10/48)} = \frac{3/48}{10/48} = \mathbf{0.3}$

第四步：计算后验期望 $E[\theta | x]$

后验期望即为后验分布的加权平均：



$$E[\theta | x] = \sum \theta_i P(\theta_i | x)$$

$$E[\theta | x] = \left( \frac{1}{4} \times 0.3 \right) + \left( \frac{1}{2} \times 0.4 \right) + \left( \frac{3}{4} \times 0.3 \right)$$

$$E[\theta | x] = 0.075 + 0.2 + 0.225 = \mathbf{0.5}$$

## 2.求UMVUE，是否有效

### 题目：

设 $X_1, X_2, \dots, X_n$ 是来自总体 $X$ 的简单随机样本，其概率密度函数为：



$$f(x; \theta) = \begin{cases} \frac{1}{\theta}x^{\frac{1}{\theta}-1}, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$



其中参数 $\theta > 0$ 未知。

1. 求参数 $\theta$ 的一致最小方差无偏估计（UMVUE）。
2. 判断该 UMVUE 是否为有效估计（即是否达到 C-R 下界）。

### 解析

#### 第一步：寻找充分完备统计量

首先将密度函数改写为指数族形式：



$$f(x; \theta) = \frac{1}{\theta} \exp\left[ \left(\frac{1}{\theta}-1\right) \ln x \right]$$



联合密度函数为：



$$L(\theta) = \theta^{-n} \exp\left[ \left(\frac{1}{\theta}-1\right) \sum_{i=1}^n \ln X_i \right]$$



由因子分解定理可知，$T = \sum_{i=1}^n (-\ln X_i)$ 是 $\theta$ 的充分统计量。由于该分布属于指数族，其自然形式下的统计量也是完备的。

#### 第二步：求 UMVUE（可能用到分部积分）

为了方便计算，我们令 $Y_i = -\ln X_i$。

利用变量转换法（或分布函数法）：

$P(Y_i \le y) = P(-\ln X_i \le y) = P(X_i \ge e^{-y}) = 1 - (e^{-y})^{1/\theta} = 1 - e^{-y/\theta}$。

可见 $Y_i \sim Exp(\theta)$（即均值为 $\theta$ 的指数分布）。

因此，$E[Y_i] = \theta$。

由样本均值的性质：$E[\bar{Y}] = E[\frac{1}{n}\sum_{i=1}^n Y_i] = \theta$。

结论： 因为 $\bar{Y} = -\frac{1}{n}\sum_{i=1}^n \ln X_i$ 是完全充分统计量 $T$ 的函数，且是 $\theta$ 的无偏估计，根据 Lehmann-Scheffé 定理，$\theta$ 的 UMVUE 为：



$$\hat{\theta} = -\frac{1}{n}\sum_{i=1}^n \ln X_i$$

#### 第三步：判断是否有效（C-R 下界）

1. 计算 Fisher 信息量 $I(\theta)$：

   $\ln f(x; \theta) = -\ln \theta + (\frac{1}{\theta}-1) \ln x$

   $\frac{\partial \ln f}{\partial \theta} = -\frac{1}{\theta} - \frac{1}{\theta^2} \ln x$

   $\frac{\partial^2 \ln f}{\partial \theta^2} = \frac{1}{\theta^2} + \frac{2}{\theta^3} \ln x$

   $I(\theta) = -E[\frac{\partial^2 \ln f}{\partial \theta^2}] = -\frac{1}{\theta^2} - \frac{2}{\theta^3} E[\ln x]$

   已知 $E[\ln X] = -E[Y] = -\theta$，代入得：

   $I(\theta) = -\frac{1}{\theta^2} - \frac{2}{\theta^3}(-\theta) = \frac{1}{\theta^2}$

2. 计算 C-R 下界：

   $CRLB = \frac{1}{n I(\theta)} = \frac{\theta^2}{n}$

3. 计算 UMVUE 的方差：

   $Var(\hat{\theta}) = Var(\frac{1}{n}\sum Y_i) = \frac{1}{n^2} \cdot n Var(Y_i) = \frac{\theta^2}{n}$（指数分布的方差为 $\theta^2$）。

**结论：** 因为 $Var(\hat{\theta}) = CRLB$，所以该 UMVUE 是**有效估计**。

## 3.似然比检验+势函数+无偏检验（PPT例题）

（1）似然比检验

：令 $\Theta_0 = \{\mu \leq \mu_0\}$，$\Theta_1 = \{\mu > \mu_0\}$，$\Theta = \Theta_0 \cup \Theta_1$，则假设检验问题为：



$$H_0: \mu \in \Theta_0, \quad H_1: \mu \in \Theta_1$$

似然比统计量为：



$$\lambda(x) = \frac{\sup_{\theta \in \Theta} \{p(x_1, \dots, x_n, \theta)\}}{\sup_{\theta \in \Theta_0} \{p(x_1, \dots, x_n, \theta)\}}$$

- 当 $\mu \in \Theta$ 时，$\mu$ 的极大似然估计 $\hat{\mu} = \bar{x}$。

- 当 $\mu \in \Theta_0$ 时：

  - 若 $\bar{x} \leq \mu_0$，则 $\bar{x}$ 是 $\mu$ 的极大似然估计。

  - 若 $\bar{x} > \mu_0$，由于：

    

    $$\frac{\partial \ln L(\mu)}{\partial \mu} = \frac{n}{\sigma_0^2}(\bar{x} - \mu) > 0$$

    - 对数似然函数 $\ln L(\mu)$ 在 $\Theta_0$ 处是 $\mu$ 的**单调增函数**。
    - 因而 $\ln L(\mu)$ 在 $\mu_0$ 处取最大值，$\mu_0$ 是 $\mu$ 的极大似然估计。

综上所述，$\hat{\mu}_0 = \min\{\bar{x}, \mu_0\}$ 是 $\mu$ 在 $\Theta_0$ 下的极大似然估计。

将 $\hat{\mu}$ 和 $\hat{\mu}_0$ 代入似然比统计量有



$$\lambda(x_1, \dots, x_n) = \frac{(2\pi\sigma_0^2)^{-\frac{n}{2}} \exp \left\{ -\frac{1}{2\sigma_0^2} \sum_{i=1}^n (x_i - \bar{x})^2 \right\}}{(2\pi\sigma_0^2)^{-\frac{n}{2}} \exp \left\{ -\frac{1}{2\sigma_0^2} \sum_{i=1}^n (x_i - \hat{\mu}_0)^2 \right\}}$$

$$= \begin{cases} 1, & \bar{x} \leq \mu_0 \\ \exp \left\{ \frac{1}{2} \left( \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}} \right)^2 \right\}, & \bar{x} > \mu_0 \end{cases}$$

令 $z = \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}}$，则 $\lambda(x) = \begin{cases} 1, & z \leq 0 \\ \exp \{ \frac{1}{2} z^2 \}, & z > 0 \end{cases}$

因为 $\lambda(x)$ 是 $z$ 的非降函数，且 $\lambda(x) \geq 1$，所以对任意正常数 $c$，欲使**犯第一类错误的概率** $P_\mu \{ \lambda(x) \geq c \} < 1$，必须要求 $c > 1$，此时有 $z \geq c_1 > 0$。当 $\mu \in \Theta_0$ 时，由于 $\frac{\bar{x} - \mu}{\sigma_0 / \sqrt{n}} \sim N(0, 1)$，所以对于大于 1 的常数 $c$ 有

$$z = \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}}$$

$$P_\mu \{ \lambda(x) \geq c \} = P_\mu \{ z \geq c_1 \} = P_\mu \left\{ \frac{\bar{x} - \mu}{\sigma_0 / \sqrt{n}} \geq c_1 + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} \right\}$$

$$= 1 - \Phi \left( c_1 + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} \right)$$

当 $\mu \leq \mu_0$ 时，$P_\mu \{\lambda(x) \geq c\}$ 是 $\mu$ 的单增函数，故对给定的显著性水平 $\alpha$，不等式 $P_\mu \{\lambda(x) \geq c\} \leq \alpha$ 当且仅当 $\mu = \mu_0$ 时等号成立，即



$$\sup_{\mu \in \Theta_0} P_\mu \{\lambda(x) \geq c\} = P_{\mu_0} \{\lambda(x) \geq c\} = P_{\mu_0} \{z \geq c_1\} = \alpha$$

由此可得 $c_1 = z_{1-\alpha}$，这样检验统计量可取为 $z = \frac{\bar{x}-\mu}{\sigma_0 / \sqrt{n}}$，在显著性水平 $\alpha$ 下，所考虑检验问题的拒绝域为 $W = \{(x_1, x_2, \dots, x_n) : z \geq z_{1-\alpha}\}$，这是通常的单侧 $z-$ 检验

（2）势函数

在显著性水平 $\alpha$ 下，拒绝域为 $W = \{ z \geq z_{1-\alpha} \}$，其中 $z = \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}}$。

势函数计算：



$$g(\mu) = P_\mu \{ \text{拒绝 } H_0 \} = P_\mu \{ z \geq z_{1-\alpha} \}$$

$$g(\mu) = P_\mu \left\{ \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}} \geq z_{1-\alpha} \right\}$$

标准化处理：为了利用标准正态分布，我们需要减去真实的均值 $\mu$：



$$g(\mu) = P_\mu \left\{ \frac{\bar{x} - \mu}{\sigma_0 / \sqrt{n}} \geq z_{1-\alpha} + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} \right\}$$



由于 $\frac{\bar{x} - \mu}{\sigma_0 / \sqrt{n}} \sim N(0, 1)$，代入标准正态分布累积分布函数 $\Phi$：



$$g(\mu) = 1 - \Phi \left( z_{1-\alpha} + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} \right)$$

（3）

**该检验是无偏检验（Unbiased Test）。**

### 判断依据：

一个检验被称为**无偏的**，如果它在备择假设空间 $\Theta_1$ 上的势总是大于或等于它在原假设空间 $\Theta_0$ 上的最大势（即显著性水平 $\alpha$）。

1. **在 $H_0$ 空间内 ($\mu \leq \mu_0$)**：
   - 势函数 $g(\mu)$ 是 $\mu$ 的单调增函数。
   - 当 $\mu \leq \mu_0$ 时，$g(\mu) \leq g(\mu_0) = \alpha$。
2. **在 $H_1$ 空间内 ($\mu > \mu_0$)**：
   - 当 $\mu > \mu_0$ 时，由于 $\frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}}$ 为负数，则 $z_{1-\alpha} + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} < z_{1-\alpha}$。
   - 因此，$\Phi \left( z_{1-\alpha} + \frac{\mu_0 - \mu}{\sigma_0 / \sqrt{n}} \right) < \Phi(z_{1-\alpha}) = 1 - \alpha$。
   - 最终得到 $g(\mu) > 1 - (1 - \alpha) = \alpha$。

**总结**：对于所有的 $\mu \in \Theta_1$，都有 $g(\mu) > \alpha$；对于所有的 $\mu \in \Theta_0$，都有 $g(\mu) \leq \alpha$。这完全符合无偏检验的定义。
