# 数理统计知识多叉树

# 参数估计

```
参数估计
├─ 一、数理统计基础
│  ├─ 核心定义：总体、个体、样本、简单随机抽样
│  ├─ 研究内容：试验设计（数据获取）、数据整理与推断
│  └─ 应用场景：各类随机现象的规律分析
├─ 二、统计量简介
│  ├─ 定义：样本的无未知参数函数（提取总体信息）
│  ├─ 常用统计量
│  │  ├─ 描述集中趋势：样本均值、样本中位数
│  │  ├─ 描述分散程度：样本方差、样本标准差、样本极差
│  │  ├─ 矩类：样本原点矩、样本中心矩
│  │  └─ 顺序统计量：排序后的样本值（最小值、最大值等）
│  └─ 特殊统计量：充分统计量（压缩数据不丢信息）、最小充分统计量（最优压缩）
├─ 三、参数估计方法（步骤）
│  ├─ 1. 频率估计
│  │  ├─ 原理：用样本频率替换总体概率
│  │  ├─ 特点：估计可能不唯一
│  │  ├─ 求解步骤
│  │  │  1. 确定总体中各结果的概率表达式（如p_i=h_i(θ)）
│  │  │  2. 计算样本中各结果的发生频率n_i/n（n_i为结果i的发生次数）
│  │  │  3. 用频率替换概率，代入概率表达式反解参数θ的估计
│  │  └─ 实例：正态分布参数估计、Hardy-Weinberg模型参数估计
│  ├─ 2. 矩估计
│  │  ├─ 原理：样本矩依概率收敛于总体矩，用样本矩替换总体矩
│  │  ├─ 适用条件：总体存在适当阶矩
│  │  ├─ 特点：直观简便、适用面广，未充分利用分布信息
│  │  ├─ 求解步骤
│  │  │  1. 计算总体的前s阶原点矩μ_k=E(X^k)=g_k(θ₁,θ₂,…,θₛ)（s为待估参数个数）
│  │  │  2. 建立方程组：μ₁=g₁(θ₁,…,θₛ)、μ₂=g₂(θ₁,…,θₛ)、…、μₛ=gₛ(θ₁,…,θₛ)
│  │  │  3. 求解方程组，将θ_j表示为总体矩的函数θ_j=h_j(μ₁,…,μₛ)
│  │  │  4. 用样本原点矩A_k=∑X_i^k/n替换总体矩μ_k，得到θ_j的矩估计θ̂_j=h_j(A₁,…,Aₛ)
│  │  └─ 实例：总体均值与方差估计、均匀分布参数估计
│  └─ 3. 极大似然估计（MLE）
│     ├─ 原理：样本发生概率最大时的参数为估计值
│     ├─ 求解步骤（★核心步骤题）
│     │  1. 构造似然函数：若总体密度/分布律为p(x;θ)，则L(θ)=∏p(x_i;θ)（样本独立同分布）
│     │  2. 取对数似然函数：lnL(θ)=∑lnp(x_i;θ)（简化求导计算）
│     │  3. 求导找极值点：对θ的各分量求偏导，令∂lnL(θ)/∂θ_j=0，解似然方程组
│     │  4. 验证极值：通过二阶导数或实际意义判断是否为最大值点（如均匀分布直接分析边界）（判断极大值能否取到）
│     │  5. 得到MLE：若似然方程组有解则为估计值，无解则结合分布特性确定（如顺序统计量）
│     ├─ 特点：充分利用分布信息，是充分统计量的函数
│     └─ 实例：二项分布、正态分布、均匀分布、Poisson分布参数估计
├─ 四、估计优良性评判（含步骤类考点）
│  ├─ 1. 均方误差（MSE）准则
│  │  ├─ 定义：MSE=方差+偏差²
│  │  ├─ 计算步骤（★）
│  │  │  1. 计算估计量的期望E(T)=q(θ)+b(θ)（b(θ)为偏差）
│  │  │  2. 计算估计量的方差Var(T)=E(T²)-[E(T)]²
│  │  │  3. 代入MSE公式：MSE=Var(T)+b²(θ)
│  │  └─ 核心：选择MSE较小的估计
│  ├─ 2. 无偏性
│  │  ├─ 定义：估计量期望=真实参数（偏差=0）
│  │  ├─ 验证步骤（★）
│  │  │  1. 写出估计量T(X₁,…,Xₙ)的表达式
│  │  │  2. 计算期望E(T)，化简后判断是否等于待估参数q(θ)
│  │  ├─ 特点：可能不存在或不唯一，无偏未必最优
│  │  └─ 实例：样本均值是总体均值的无偏估计，样本方差是总体方差的无偏估计
│  ├─ 3. 一致最小方差无偏估计（UMVUE）
│  │  ├─ 定义：无偏估计类中方差最小
│  │  ├─ 求解步骤（★基于Lehmann-Scheffe定理）
│  │  │  1.寻找 完全充分统计量S(X) （通过因式分解定理或指数分布族性质）
│  │  │  2.构造待估参数q(θ)的一个 无偏估计φ(X)
│  │  │  3.计算条件期望T(X)=E(φ(X)|S(X))(一般用概率算)，得到UMVUE（若φ(X)已是S(X)的函数，则φ(X)即为UMVUE）
│  │  ├─ 存在性与唯一性：满足特定条件时存在且唯一
│  │  └─ 实例：正态总体均值（样本均值）、方差（样本方差）的UMVUE
│  ├─ 4. 信息不等式（Cramer-Rao不等式）
│  │  ├─ 核心：无偏估计的方差存在下界（C-R下界）
│  │  ├─ 计算步骤（★）
│  │  │  1. 计算Fisher信息量I(θ)=E[(∂lnp(X;θ)/∂θ)²]（或-I(θ)=E[∂²lnp(X;θ)/∂θ²]）
│  │  │  2. 样本Fisher信息量Iₙ(θ)=nI₁(θ)（I₁为单样本信息量）
│  │  │  3. 计算C-R下界：1/Iₙ(θ)，[q’(θ)]²/Iₙ(θ)
│  │  │  4. 判断有效性：若估计量方差等于下界则为有效估计
│  │  ├─ 相关概念：Fisher信息量（衡量参数信息含量）
│  │  └─ 特殊估计：有效估计（方差达到下界）、渐进有效估计（大样本下有效率趋近1）
│  └─ 5. 相合性
│     ├─ 定义：大样本下依概率收敛于真实参数（大样本性质）
│     ├─ 证明步骤
│     │  1. 方法一（利用定义）：对任意ε>0，证明limₙ→∞P(|Tₙ-q(θ)|>ε)=0
│     │  2. 方法二（利用大数定律）：若Tₙ是样本矩的连续函数，且样本矩依概率收敛于总体矩，则Tₙ相合
│     │  3. 方法三（利用Chebyshev不等式）：证明limₙ→∞MSE(Tₙ)=0，则Tₙ相合
│     ├─ 相关性质：连续函数保持相合性
│     └─ 补充：渐进无偏估计（大样本下期望趋近真实值）、渐进正态性（大样本服从正态分布）
└─ 五、重要定理与工具（支撑步骤题求解）
   ├─ 因式分解定理：判断充分统计量的充要条件（步骤题中用于找充分统计量）
   ├─ 大数定律：样本矩收敛于总体矩的理论依据（矩估计、相合性证明基础）
   ├─ 常见分布：χ²分布、t分布、F分布（统计推断基础，用于方差、期望估计的分布验证）
   └─ 求解方法：似然方程、条件期望、牛顿法等（MLE、UMVUE求解的辅助工具）
```

# 假设检验

```
假设检验
├─ 一、基础概念
│  ├─ 核心定义
│  │  ├─ 原假设（H₀）与备择假设（H₁）：简单假设vs复合假设
│  │  ├─ 检验统计量：构造依据（如样本均值、似然比）
│  │  ├─ 拒绝域（W）与接受域（W^c）：样本空间的划分
│  │  ├─ 检验函数：非随机化检验（0-1取值）vs随机化检验（[0,1]取值）
│  │  └─ 显著性水平（α）：控制第一类错误的上限
│  ├─ 两类错误与势函数
│  │  ├─ 第一类错误：H₀为真时拒绝H₀，概率α(θ)（去真）
│  │  ├─ 第二类错误：H₀为假时接受H₀，概率β(θ)（存伪）
│  │  └─ 势函数（g(θ)）：H₀不成立时拒绝H₀的概率，g(θ)=1-β(θ)
│  └─ 核心原理：Neyman-Pearson检验原理（控制第一类错误，最大化检验势）
├─ 二、假设检验通用步骤（★简答题高频考点）
│  1. 提出假设：明确H₀（θ∈Θ₀）与H₁（θ∈Θ₁）
│  2. 选取检验统计量：基于H₀的分布特性确定（如Z统计量、T统计量）
│  3. 给定显著性水平α（常用0.01、0.05、0.1）
│  4. 确定拒绝域：根据统计量分布和α计算临界值，划分W与W^c
│  5. 计算统计量值：代入样本观测值求解
│  6. 作出推断：统计量∈W则拒绝H₀，否则不拒绝H₀
├─ 三、核心检验方法（含步骤类考点）
│  ├─ 1. 似然比检验（★步骤类简答题）
│  │  ├─ 核心思想：比较H₁与H₀下的似然函数比值，比值过大则拒绝H₀
│  │  ├─ 通用步骤（★）
│  │  │  1. 分别在Θ（全（或者补）参数空间）和Θ₀（H₀参数空间）中求θ的极大似然估计（Ĥ、Ĥ₀）
│  │  │  2. 构造似然比统计量λ(x)=p(x,Ĥ)/p(x,Ĥ₀)（λ(x)≥1）
│  │  │  3. 化简λ(x)为h(T(x))，要求h是T(x)的单调函数，且H₀成立时T(x)分布已知
│  │  │  4. 确定临界值：**单调**增则由P(T≥c₁|H₀)=α求c₁，单调减则由P(T≤c₁|H₀)=α求c₁
│  │  │  5. 确定拒绝域：单调增对应W={T≥c₁}，单调减对应W={T≤c₁}
│  │  └─ 实例：正态总体（方差已知/未知）、离散总体的似然比检验
│  ├─ 2. Neyman-Pearson引理（MPT构造依据）
│  │  ├─ 适用场景：简单H₀（θ=θ₀）vs简单H₁（θ=θ₁）
│  │  ├─ 核心结论：最优势检验（MPT）的拒绝域由似然比L(x)=p(x,θ₁)/p(x,θ₀)确定
│  │  └─ MPT检验函数（★步骤类）
│  │     1. 连续型L(x)：φ(x)=1（L(x)>k）、0（L(x)<k），k由P(L(x)≥k|H₀)=α确定
│  │     2. 离散型L(x)：φ(x)=1（L(x)>k）、r（L(x)=k）、0（L(x)<k），r=(α-P(L(x)>k|H₀))/P(L(x)=k|H₀)
（就是通过让期望=α）
│  ├─ 3. 一致最优势检验（UMPT）
│  │  ├─ 定义：对所有θ∈Θ₁，势函数均不小于同水平其他检验
│  │  ├─ 存在条件：样本联合密度满足**指数分布族**形式p(x,θ)=d(θ)h(x)exp{c(θ)T(x)}，c(θ)严格单调
│  │  ├─ 构造步骤（★单边检验，θ≤θ₀vsθ>θ₀）
│  │  │  1. 验证样本密度符合指数分布族，确定T(x)和c(θ)的单调性（一般c(θ)增）
│  │  │  2. 构造检验函数：φ*(x)=1（T(x)>c（拒绝区域））、r（T(x)=c）、0（T(x)<c）
│  │  │  3. 确定c和r：由E_{θ₀}(φ*(x))=α求解
│  │  └─ 实例：指数分布、正态分布、Poisson分布的UMPT构造
│  └─ 4. 一致最优势无偏检验（UMPUT）
│     ├─ 适用场景：双边检验（H₀:θ=θ₀vsH₁:θ≠θ₀），UMPT不存在时使用
│     ├─ 无偏检验定义：势函数g(θ)≤α（θ∈Θ₀）且g(θ)≥α（θ∈Θ₁）
│     ├─ 构造步骤（★）
│     │  1. 验证样本密度符合指数分布族，确定T(x)和c(θ)的单调性
│     │  2. 构造检验函数：φ*(x)=1（T(x)<c₁或T(x)>c₂）、rᵢ（T(x)=cᵢ）、0（c₁<T(x)<c₂）（拒绝时取1）
│     │  3. 确定常数：双边检验H₀:θ=θ₀时，由E_{θ₀}(φ*(x))=α和E_{θ₀}(T(x)φ*(x))=αE_{θ₀}(T(x))求解c₁、c₂、r₁、r₂
│     └─ 实例：正态总体均值的双边u检验（UMPUT）
├─ 四、检验类型分类
│  ├─ 按假设形式
│  │  ├─ 单边检验：H₀:θ≤θ₀vsH₁:θ>θ₀；H₀:θ≥θ₀vsH₁:θ<θ₀
│  │  └─ 双边检验：H₀:θ=θ₀vsH₁:θ≠θ₀；H₀:θ₁≤θ≤θ₂vsH₁:θ<θ₁或θ>θ₂

│  └─ 按总体类型
│     ├─ 正态总体：均值检验（u检验、t检验）、方差检验（χ²检验）
│     └─ 非正态总体：指数分布、Poisson分布、离散分布的检验
└─ 五、关键定理与性质
   ├─ 势函数性质：样本容量固定时，两类错误概率不可同时减小；增大样本容量可同时减小
   ├─ UMPT存在定理：指数分布族+单边检验时UMPT存在
   ├─ UMPUT存在定理：指数分布族+双边检验时UMPUT存在
   └─ 似然比统计量极限分布：大样本下2lnλ(x)~χ²(自由度=dimΘ-dimΘ₀)
```

$$E_{\theta_0}(\phi(x)) = \alpha$$

控制第一类错误



设功效函数为 $g(\theta) = E_\theta(\phi(x))$。对于双边检验：

- 在 $\theta = \theta_0$ 时，$g(\theta_0) = \alpha$。
- 如果检验是非偏的，那么对于任何 $\theta \neq \theta_0$，都有 $g(\theta) \ge \alpha$。

原理： $g(\theta)$ 在 $\theta = \theta_0$ 处必须取得**最小值**。

$f(x;\theta) = C(\theta) \exp(\theta T(x)) h(x)$ 下，我们对功效函数求导：



$$g'(\theta) = \frac{d}{d\theta} \int \phi(x) f(x;\theta) dx$$

条件：$$E_{\theta_0}(T(x)\phi(x)) = \alpha E_{\theta_0}(T(x))$$

# 贝叶斯

```
贝叶斯分析
├─ 一、基础概念与核心思想
│  ├─ 核心定义
│  │  ├─ 三大学派：频率学派（依赖总体+样本信息）、贝叶斯学派（新增先验信息）、其他学派
│  │  ├─ 三种信息：总体信息（分布族）、样本信息（观测数据）、先验信息（历史/经验/理论）
│  │  ├─ 关键分布：先验分布π(θ)（观测前对θ的信念）、似然函数p(x|θ)（θ给定下x的概率）、后验分布π(θ|x)（融合三种信息后的信念）
│  │  └─ 核心分歧：贝叶斯学派将θ视为随机变量，频率学派视θ为未知参数
│  ├─ 贝叶斯公式（核心工具）
│  │  ├─ 连续型：π(θ|x)=[p(x|θ)π(θ)]/∫p(x|θ)π(θ)dθ，其中分母m(x)为x的边缘密度（无θ信息）
│  │  └─ 离散型：π(θ_i|x)=[p(x|θ_i)π(θ_i)]/∑p(x|θ_j)π(θ_j)
│  └─ 核心思想：通过观测数据更新先验信念，得到后验信念，作为推断依据
├─ 二、核心方法（★步骤简答题高频考点）
│  ├─ 1. 后验分布计算（★核心步骤题）
│  │  ├─ 适用场景：已知先验分布和样本，求参数θ的后验分布
│  │  ├─ 求解步骤（★）
│  │  │  1. 确定似然函数p(x|θ)：根据总体分布，写出样本联合密度（独立同分布时为乘积形式）
│  │  │  2. 指定先验分布π(θ)：基于先验信息（如无信息时用均匀分布、贝塔分布等共轭先验）
│  │  │  3. 计算联合分布h(x,θ)=p(x|θ)π(θ)：融合三种信息
│  │  │  4. 计算边缘密度m(x)=∫h(x,θ)dθ（连续型）或∑h(x,θ_i)（离散型）：归一化因子
│  │  │  5. 代入贝叶斯公式：得到后验分布π(θ|x)=h(x,θ)/m(x)
│  │  └─ 实例：二项分布+均匀先验→贝塔后验、离散参数（θ₁/θ₂）的后验概率更新
│  ├─ 2. 贝叶斯估计（★步骤题）
│  │  ├─ 原理：最小化期望风险，常用均方误差损失函数λ(θ,â)=(θ-â)²
│  │  ├─ 求解步骤（★）
│  │  │  1. 按“后验分布计算步骤”得到π(θ|D)（D为样本集）
│  │  │  2. 若采用均方误差损失，贝叶斯估计量为后验期望：â=E(θ|D)=∫θπ(θ|D)dθ（连续型）
│  │  │  3. 代入后验分布表达式，计算积分或求和，化简得到估计结果
│  │  ├─ 特点：融合先验信息，数据量增大时先验影响减弱；均匀先验等价于经典方法
│  │  └─ 实例：二项分布参数θ的贝叶斯估计、正态分布均值μ的贝叶斯估计（已知方差）
│  └─ 3. 贝叶斯网络构建与推理
│     ├─ 核心概念：有向无环图（DAG）+局部条件概率分布，刻画变量间条件独立关系
│     ├─ 网络构建步骤（★）
│     │  1. 确定随机变量集合X={X₁,X₂,…,Xₙ}：刻画问题的关键变量
│     │  2. 选择变量顺序：优先按因果关系（原因在前，结果在后）或降低模型复杂度
│     │  3. 逐个添加变量，确定父节点π(Xᵢ)：满足P(Xᵢ|X₁,…,Xᵢ₋₁)=P(Xᵢ|π(Xᵢ))（条件独立）
│     │  4. 为每个节点赋值：根节点为边缘分布P(Xᵢ)，非根节点为条件概率分布P(Xᵢ|π(Xᵢ))
│     ├─ 联合分布分解：P(X₁,…,Xₙ)=∏P(Xᵢ|π(Xᵢ))（利用条件独立简化）
│     ├─ 概率推理步骤（★）
│     │  1. 明确目标：计算目标变量的后验概率（如P(B=y|M=y)）
│     │  2. 利用分解式写出联合分布，代入已知证据（如M=y）
│     │  3. 对无关变量求和/积分：计算目标变量与证据的联合概率P(目标,证据)
│     │  4. 计算证据的边缘概率P(证据)：对所有目标变量取值求和/积分
│     │  5. 后验概率=联合概率/边缘概率：得到推理结果
│     └─ 实例：Alarm问题（盗窃/地震→警铃→电话）的网络构建与后验推理
├─ 三、进阶主题：因果推断（选讲）
│  ├─ 核心概念：相关性≠因果性、潜在结果模型（Y(1)/Y(0)）、平均因果效应（ACE=E[Y(1)-Y(0)]）
│  ├─ 后门调整法（★因果推断步骤题）
│  │  ├─ 适用场景：存在混淆变量C，需控制C以识别因果效应
│  │  ├─ 步骤（★）
│  │  │  1. 绘制因果图：明确目标变量（X=干预，Y=结果）和混淆变量C
│  │  2. 验证后门路径：确认C阻断所有X→Y的后门路径（非因果路径）
│  │  3. 应用后门公式：P(Y|do(X))=∑c P(Y|X,c)P(C=c)
│  │  4. 代入观测数据中的P(Y|X,c)和P(C=c)，计算得到干预后的分布
│  │  5. 计算ACE：ACE=E[Y|do(X=1)]-E[Y|do(X=0)]
│  │  └─ 实例：辛普森悖论（药品A/B+年龄混淆）的因果效应校正
│  └─ 关键工具：Do算子（干预）、截断因子分解（固定干预变量后的分布）
└─ 四、重要定理与性质
   ├─ 共轭先验：先验与后验属于同一分布族，简化计算（如二项分布→贝塔分布、正态分布→正态分布）
   ├─ 局部马尔可夫性：每个节点给定父节点后，与非后代节点条件独立
   ├─ 马尔可夫毯：节点的父节点、子节点、子节点的父节点，给定其则与其他节点独立
   └─ 贝叶斯网络的全局语义：联合分布=局部条件概率分布的乘积
```

# 多元统计基础

```
多元统计基础
├─ 一、核心概念
│  ├─ 1. 随机向量与样本资料矩阵
│  │  ├─ 随机向量：p个随机变量的有序组合X=(X₁,X₂,…,Xₚ)ᵀ，分离散型和连续型
│  │  ├─ 样本资料矩阵：n行（样本数）×p列（变量数），每行是随机向量的一次观测
│  │  └─ 核心关联：样本资料矩阵是随机向量的观测实现，列数对应变量数p
│  ├─ 2. 分布函数与密度函数
│  │  ├─ 多元分布函数F(x)：P(X₁≤x₁,…,Xₚ≤xₚ)，取值[0,1]，满足单调性、有界性
│  │  ├─ 多元密度函数f(x)：连续型随机向量专属，F(x)是f(x)的p重积分，f(x)≥0且全空间积分=1
│  │  └─ 关联关系：f(x)是F(x)的p阶混合偏导数（连续点处）
│  ├─ 3. 独立性与数字特征
│  │  ├─ 随机向量独立：联合分布=各边缘分布乘积（分布函数/密度函数均成立）
│  │  ├─ 均值向量：μ=E(X)=(E(X₁),…,E(Xₚ))ᵀ，线性性质E(AX+b)=Aμ+b
│  │  ├─ 协方差阵：Σ=D(X)=E[(X-μ)(X-μ)ᵀ]，对称半正定（多数正定），性质D(AX)=AΣAᵀ，对角线元素
│  │  ├─ 相关阵：R=(rᵢⱼ)，rᵢⱼ=cov(Xᵢ,Xⱼ)/(√D(Xᵢ)√D(Xⱼ))，标准化数据的协方差阵=原相关阵
│  │  └─ 协方差阵与相关性：cov(X,Y)=0→线性不相关，正态分布下不相关=独立
├─ 二、多元正态分布（核心分布）
│  ├─ 1. 定义（三种等价形式）
│  │  ├─ 密度函数定义：f(x)=(2π)⁻ᵖ/²|V|⁻¹/²exp{-½(X-μ)ᵀV⁻¹(X-μ)}，V>0
│  │  ├─ 特征函数定义：φ(t)=exp{itᵀμ - ½tᵀVt}，V≥0
│  │  └─ 线性组合定义：对任意非零a∈Rᵖ，aᵀX~N(aᵀμ,aᵀVa)
│  ├─ 2. 核心性质（★简答题高频）
│  │  ├─ 唯一性：由均值向量μ和协方差阵V唯一确定，记为X~Nₚ(μ,V)
│  │  ├─ 线性变换不变性：Y=AX+b~Nₘ(Aμ+b,AΣAᵀ)（A为m×p常数阵）
│  │  ├─ 独立性等价：正态向量分块后，互不相关=相互独立
│  │  ├─ 平方分布：V>0时，(X-μ)ᵀV⁻¹(X-μ)~χ²(p)
│  │  └─ 条件分布：分块X=(X₁ᵀ,X₂ᵀ)ᵀ，给定X₂时X₁~Nₖ(μ₁·₂,V₁₁·₂)，其中μ₁·₂=μ₁+V₁₂V₂₂⁻¹(X₂-μ₂)，V₁₁·₂=V₁₁-V₁₂V₂₂⁻¹V₂₁
│  ├─ 3. 条件分布与偏相关
│  │  ├─ 条件期望E(X₁|X₂)：X₁对X₂的多元回归函数，回归系数矩阵=V₁₂V₂₂⁻¹
│  │  ├─ 偏相关系数：扣除其他变量影响后，两变量的相关系数，rᵢⱼ·ₖ₊₁…ₚ=σᵢⱼ·ₖ₊₁…ₚ/(√σᵢᵢ·ₖ₊₁…ₚ√σⱼⱼ·ₖ₊₁…ₚ)
│  └─ 实例：分块正态向量的条件分布计算、回归函数求解
├─ 三、参数估计（★步骤简答题高频考点）
│  ├─ 估计对象：多元正态总体Nₚ(μ,V)的均值向量μ和协方差阵V（V>0，n>p）
│  ├─ 1. 极大似然估计（MLE）步骤（★核心步骤题）
│  │  1. 给定简单样本X₁,X₂,…,Xₙ，构造样本均值向量：μ̂=X̄=(1/n)ΣXᵢ
│  │  2. 计算样本离差阵：S=Σ(Xᵢ-X̄)(Xᵢ-X̄)ᵀ
│  │  3. 求解MLE：μ的MLE=X̄，V的MLE=(1/n)S
│  ├─ 2. 无偏估计（★步骤题）
│  │  1. 均值向量无偏估计：μ̂=X̄（E(X̄)=μ）
│  │  2. 协方差阵无偏估计：Σ̂=(1/(n-1))S（E((1/(n-1))S)=V）
│  ├─ 估计量性质
│  │  ├─ X̄~Nₚ(μ,V/n)，且X̄与S相互独立
│  │  ├─ X̄是μ的UMVUE，(1/(n-1))S是V的UMVUE
│  │  └─ S~Wₚ(n-1,V)（Wishart分布，多元χ²分布推广）
├─ 四、均值向量的检验（★步骤简答题核心）
│  ├─ 检验类型：单总体均值检验（H₀:μ=μ₀）、两总体均值相等检验（H₀:μ₁=μ₂）
│  ├─ 1. 单总体均值检验
│  │  ├─ 情形1：协方差阵V已知
│  │  │  1. 提出假设：H₀:μ=μ₀，H₁:μ≠μ₀
│  │  │  2. 构造检验统计量：D=n(X̄-μ₀)ᵀV⁻¹(X̄-μ₀)（H₀成立时D~χ²(p)）
│  │  │  3. 给定显著性水平α，确定临界值：χ₁₋α²(p)
│  │  │  4. 计算统计量D，拒绝域：W={D≥χ₁₋α²(p)}
│  │  │  5. 推断：D∈W则拒绝H₀，否则不拒绝
│  │  ├─ 情形2：协方差阵V未知
│  │  │  1. 提出假设：H₀:μ=μ₀，H₁:μ≠μ₀
│  │  │  2. 构造检验统计量：F=[n(n-p)/p](X̄-μ₀)ᵀS⁻¹(X̄-μ₀)（H₀成立时F~F(p,n-p)）
│  │  │  3. 给定显著性水平α，确定临界值：F₁₋α(p,n-p)
│  │  │  4. 计算统计量F，拒绝域：W={F≥F₁₋α(p,n-p)}
│  │  │  5. 推断：F∈W则拒绝H₀，否则不拒绝
│  ├─ 2. 两总体均值相等检验
│  │  ├─ 前提：两样本独立，X~Nₚ(μ₁,V)，Y~Nₚ(μ₂,V)（同协方差阵）
│  │  ├─ 情形1：V已知
│  │  │  1. 提出假设：H₀:μ₁=μ₂，H₁:μ₁≠μ₂
│  │  │  2. 构造检验统计量：D*=[n₁n₂/(n₁+n₂)](X̄-Ȳ)ᵀV⁻¹(X̄-Ȳ)（H₀成立时D*~χ²(p)）
│  │  │  3. 给定α，确定临界值χ₁₋α²(p)，拒绝域W={D*≥χ₁₋α²(p)}
│  │  │  4. 计算D*并推断
│  │  ├─ 情形2：V未知
│  │  │  1. 提出假设：H₀:μ₁=μ₂，H₁:μ₁≠μ₂
│  │  │  2. 计算联合协方差阵：Σ̂=[(n₁-1)Σ̂₁+(n₂-1)Σ̂₂]/(n₁+n₂-2)（Σ̂₁、Σ̂₂为两样本协方差阵）
│  │  │  3. 构造检验统计量：F*=[n₁n₂(n₁+n₂-p-1)/(p(n₁+n₂)(n₁+n₂-2))](X̄-Ȳ)ᵀΣ̂⁻¹(X̄-Ȳ)（H₀成立时F*~F(p,n₁+n₂-p-1)）
│  │  │  4. 给定α，确定临界值F₁₋α(p,n₁+n₂-p-1)，拒绝域W={F*≥临界值}
│  │  │  5. 计算F*并推断
└─ 五、重要分布与工具
   ├─ Wishart分布：多元正态样本的离差阵分布，S~Wₚ(n-1,V)，是一维χ²分布的推广
   ├─ 似然比检验：均值检验的理论基础，通过比较H₀和全参数空间的似然函数最大值构造统计量
   └─ 线性变换：多元统计分析的核心工具，用于简化分布、构造检验统计量
```

边缘分布，对其它所有变量积分

先求$\hat{μ}$，代入后求$\hat{V}$

| **估计类型**         | **一维分布 (μ)**                            | **多维分布 (μ)**                                             | **统计性质**                                |
| -------------------- | ------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------- |
| **矩估计 (MME)**     | $\hat{\mu} = \bar{X} = \frac{1}{n}\sum X_i$ | $\hat{\boldsymbol{\mu}} = \bar{\mathbf{X}} = \frac{1}{n}\sum \mathbf{X}_i$ | 简单直观，通常是无偏的。                    |
| **极大似然 (MLE)**   | $\hat{\mu}_{ML} = \bar{X}$                  | $\hat{\boldsymbol{\mu}}_{ML} = \bar{\mathbf{X}}$             | 渐近正态，是 **无偏** 且一致的。            |
| **最优估计 (UMVUE)** | $\bar{X}$ (正态分布下)                      | $\bar{\mathbf{X}}$ (多元正态下)                              | 在正态假设下达到 C-R 下界，是最优无偏估计。 |

| **估计类型**            | **一维方差 (σ2)**                                         | **多维协方差矩阵 (Σ)**                                       | **统计性质 (重点)**                                |
| ----------------------- | --------------------------------------------------------- | ------------------------------------------------------------ | -------------------------------------------------- |
| **极大似然 (MLE)**      | $\hat{\sigma}^2_{ML} = \frac{1}{n}\sum (X_i - \bar{X})^2$ | $\hat{\boldsymbol{\Sigma}}_{ML} = \frac{1}{n}\sum (\mathbf{X}_i - \bar{\mathbf{X}})(\mathbf{X}_i - \bar{\mathbf{X}})^T$ | **有偏估计**。倾向于低估真实的变异程度。           |
| **无偏估计 (Unbiased)** | $S^2 = \frac{1}{n-1}\sum (X_i - \bar{X})^2$               | $\mathbf{S} = \frac{1}{n-1}\sum (\mathbf{X}_i - \bar{\mathbf{X}})(\mathbf{X}_i - \bar{\mathbf{X}})^T$ | **无偏**。通过将分母改为 $n-1$ 修正了 MLE 的偏差。 |
| **分布特性**            | $\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$              | $(n-1)\mathbf{S} \sim W_p(n-1, \boldsymbol{\Sigma})$         | 一维服从卡方分布，多维对应 **Wishart 分布**。      |

无偏估计推导

在 $H_0: \boldsymbol{\mu} = \boldsymbol{\mu}_0$ 成立的条件下：



$$\bar{\mathbf{X}} \sim N_p(\boldsymbol{\mu}_0, \frac{1}{n}\mathbf{V})$$

标准化变化

$D = n(\bar{\mathbf{X}}-\boldsymbol{\mu}_0)^T \mathbf{V}^{-1}(\bar{\mathbf{X}}-\boldsymbol{\mu}_0) \sim \chi^2(p)$

在多维情况下，样本协方差矩阵 $\mathbf{S}$ 实际上承载了剩余的自由度。对于一维情况，我们知道 $(n-1)S^2/\sigma^2 \sim \chi^2(n-1)$；在多维情况下，虽然中间跳过了复杂的 Wishart 分布推导，但最终通过线性变换，我们会析出一个自由度为 **$n-p$** 的卡方分布。

一维时，$T$ 检验的自由度是 $n-1$。

多维时，由于我们要估计 $p$ 个维度的均值，且 $\mathbf{S}$ 是一个 $p \times p$ 的矩阵，这种“估计”消耗了更多的自由度。

这是多维统计中最容易混淆点。

- 一维时，$T$ 检验的自由度是 $n-1$。
- 多维时，由于我们要估计 $p$ 个维度的均值，且 $\mathbf{S}$ 是一个 $p \times p$ 的矩阵，这种“估计”消耗了更多的自由度。
- 你可以这样感性理解：你需要至少 $p+1$ 个样本才能形成一个有体积的“散点包围圈”来估计协方差。损失的自由度与维数 $p$ 成正比，最终留给分母的自由度就是 $n-p$。

# 非参数统计基础

```
非参数统计基础
├─ 一、核心概念与特点
│  ├─ 定义：不依赖总体分布类型，仅利用样本秩、频率等信息进行推断
│  ├─ 适用场景：总体分布未知、样本量小、数据有结（重复值）、不满足参数检验假设
│  └─ 核心优势：稳健性强、适用范围广；劣势：信息利用不充分、功效较低
├─ 二、Pearson检验（χ²拟合检验）
│  ├─ 1. 总体分布拟合检验（检验H₀：F(x)=F₀(x)）(是否服从分布)
│  │  ├─ 求解步骤
│  │  │  1. 提出假设：H₀：总体服从给定分布F₀(x)，H₁：不服从
│  │  │  2. 划分区间：将(-∞,+∞)分成k个互斥区间A₁,A₂,…,Aₖ，保证nP(Aᵢ)≥5（否则合并区间）
│  │  │  3. 计算概率/理论频数：若F₀(x)无未知参数，Pᵢ=F₀(aᵢ₊₁)-F₀(aᵢ)；若有γ个未知参数，先求MLE代入得P̂ᵢ，理论频数=nPᵢ（或nP̂ᵢ）
│  │  │  4. 统计实际频数：计数样本落在各区间的实际次数fᵢ
│  │  │  5. 构造检验统计量：χ²=Σ[(fᵢ - nPᵢ)²/(nPᵢ)]（无未知参数）或χ²=Σ[(fᵢ - nP̂ᵢ)²/(nP̂ᵢ)]（有未知参数）
│  │  │  6. 确定自由度与临界值：无未知参数→df=k-1；有γ个未知参数→df=k-γ-1，查χ₁₋α²(df)
│  │  │  7. 推断：χ²≥χ₁₋α²(df)则拒绝H₀，否则接受
│  │  └─ 实例：检验样本是否服从Poisson分布（含未知参数λ的MLE估计）
│  ├─ 2. 列联表独立性检验（检验H₀：X与Y独立）
│  │  ├─ 求解步骤
│  │  │  1. 提出假设：H₀：X与Y相互独立，H₁：不独立
│  │  │  2. 构建r×s列联表：记录交叉分类的频数nᵢⱼ（i=1…r为X的类别，j=1…s为Y的类别）
│  │  │  3. 计算边缘频数与估计概率：nᵢ·=Σnᵢⱼ，n·ⱼ=Σnᵢⱼ，P̂ᵢ·=nᵢ·/n，P̂·ⱼ=n·ⱼ/n
│  │  │  4. 构造检验统计量：χ²=ΣΣ[(nᵢⱼ - nP̂ᵢ·P̂·ⱼ)²/(nP̂ᵢ·P̂·ⱼ)]=ΣΣ[(nᵢⱼ - nᵢ·n·ⱼ/n)²/(nᵢ·n·ⱼ/n)]
│  │  │  5. 确定自由度与临界值：df=(r-1)(s-1)，查χ₁₋α²(df)
│  │  │  6. 推断：χ²≥χ₁₋α²(df)则拒绝H₀，认为X与Y相关
│  │  └─ 实例：2×2列联表检验慢性气管炎与吸烟的独立性
│  └─ 关键注意：样本量n≥50，理论频数nPᵢ≥5，否则合并区间保证检验有效性
├─ 三、秩相关检验
│  ├─ 1. 秩统计量基础
│  │  ├─ 无结数据的秩：样本中不超过Xᵢ的数据个数Rᵢ，R=(R₁,…,Rₙ)均匀分布于(1,…,n)的排列
│  │  ├─ 有结数据的秩：重复数据组成“结”，取原秩的平均值作为结内各数据的秩
│  │  └─ 秩的性质：E(Rᵢ)=(n+1)/2，Var(Rᵢ)=(n²-1)/12，Cov(Rᵢ,Rⱼ)=-(n+1)/12（无结情形）
│  ├─ 2. 秩和检验（Wilcoxon检验，检验H₀：F(x)=G(x)）
│  │  ├─ 求解步骤
│  │  │  1. 提出假设：H₀：两总体分布相同，H₁：分布不同
│  │  │  2. 合并排序：将两样本（容量n₁≤n₂）合并，按大小排序并赋予秩（有结取平均秩）
│  │  │  3. 计算秩和：取容量较小样本的秩和T作为检验统计量
│  │  │  4. 确定临界值/分布：n₁,n₂≤10时查秩和检验表得T₁,T₂；n₁,n₂>10时，T~N(n₁(n₁+n₂+1)/2, n₁n₂(n₁+n₂+1)/12)，构造U统计量~N(0,1)
│  │  │  5. 推断：小样本时T∉(T₁,T₂)则拒绝H₀；大样本时|U|>u₁₋α/₂则拒绝
│  │  └─ 实例：两独立样本的分布一致性检验（含结数据的秩处理）
│  ├─ 3. 符号检验（配对样本，检验H₀：F(x)=G(x)）
│  │  ├─ 求解步骤
│  │  │  1. 提出假设：H₀：配对数据X,Y无差异（P(Z>0)=1/2），H₁：有差异（P(Z>0)≠1/2），其中Z=X-Y
│  │  │  2. 统计符号：计数“+”号个数n₊、“-”号个数n₋，n=n₊+n₋（忽略Z=0）
│  │  │  3. 构造统计量：X=n₊~b(n,1/2)（H₀成立时）
│  │  │  4. 确定临界值/分布：小样本查符号检验表得c,d；大样本时U=(2X-n)/√n~N(0,1)（中心极限定理）
│  │  │  5. 推断：小样本X≤d或X≥c则拒绝H₀；大样本|U|>u₁₋α/₂则拒绝
│  └─ 4. 符号秩和检验（Wilcoxon配对检验，检验H₀：F(x)=G(x)）
│     ├─ 求解步骤（★）
│     │  1. 提出假设：H₀：配对数据无差异，H₁：有差异
│     │  2. 计算差值与绝对值：Zᵢ=Xᵢ-Yᵢ，忽略Zᵢ=0，对|Zᵢ|赋予秩Rᵢ
│     │  3. 构造统计量：W⁺=Σ（Zᵢ>0时的Rᵢ）（正差值的秩和）
│     │  4. 确定临界值/分布：n≤20时查符号秩和检验表得c,d（d=n(n+1)/2 - c）；n>20时，W⁺~N(n(n+1)/4, n(n+1)(2n+1)/24)，构造标准化统计量~N(0,1)
│     │  5. 推断：W⁺≤d或W⁺≥c则拒绝H₀；大样本时|标准化统计量|>u₁₋α/₂则拒绝（W⁺取值适中）
│     └─ 实例：用户对两种产品评分的一致性检验
├─ 四、概率分布估计（非参数估计方法）
│  ├─ 1. 经验分布函数
│  │  ├─ 定义：Fₙ(x)=vₙ(x)/n，vₙ(x)为样本中≤x的个数，是阶梯函数
│  │  ├─ 性质（Glivenko定理）：n→∞时，Fₙ(x)以概率1一致收敛于理论分布F(x)
│  │  └─ 特点：直观简单，无需假设总体分布
│  └─ 2. 核密度估计（★步骤题）
│     ├─ 定义：f̂ₙ(x)=(1/(nh))ΣK((Xᵢ - x)/h)，K为核函数，h为窗宽
│     ├─ 核心步骤
│     │  1. 选择核函数：满足K(x)≥0、∫K(x)dx=1，常用高斯核、均匀核等
│     │  2. 确定窗宽h：经验公式hₒₚₜ≈1.06σ̂n⁻¹/⁵（高斯核），σ̂为样本标准差
│     │  3. 代入样本计算：将每个样本Xᵢ代入核函数，求和后除以nh得f̂ₙ(x)
│     ├─ 核函数类型：均匀核、三角核、Epanechikov核、高斯核等（单峰对称）
│     └─ 特点：光滑性优于直方图，h越小越灵敏但波动大，h越大越平滑但偏差大
└─ 五、关键工具与注意事项
   ├─ 常用分布：χ²分布（Pearson检验）、正态分布（大样本近似）
   ├─ 有结数据处理：秩平均法（秩和检验）、保留结长计算（符号秩和检验）
   └─ 样本量要求：Pearson检验n≥50，秩和检验小样本（n₁,n₂≤10）用查表法，大样本用正态近似
```
