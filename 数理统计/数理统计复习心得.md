### **2.参数估计**

### （1）求UMVUE

步骤 1：寻找完全充分统计量 (Complete Sufficient Statistic)

- 观察分布是否属于**指数族**（Exponential Family）（）通常是。

- 联合概率密度函数：形式$c(\theta)h(x_1,x_2,……,x_n)e^{w(\theta)T(x_1,x_2,……,x_n)}$,

  要求$w(\theta)$值域有内点。$T(x_1,x_2,……,x_n)$是完全充分统计量

步骤 2：构造无偏性 (Unbiasedness)

1. 计算 $T$ 的期望 $E_\theta[T]$。
2. 寻找一个函数 $g(T)$，使得 $E_\theta[g(T)] = \tau(\theta)$，其中 $\tau(\theta)$ 是你想要估计的目标参数。
3. **常用技巧**：如果直接找 $g(T)$ 困难，可以先找一个简单的无偏估计量 $W$（如单个观测值 $X_1$），然后利用 **Rao-Blackwell 定理** 求条件期望 $\hat{\theta} = E[W | T]$。

步骤 3：定论 (Conclusion)

- 根据 **Lehmann-Scheffé 定理**，一个基于完全充分统计量的无偏估计量是**唯一**的 UMVUE。

### （2）判断是否是有效估计

计算 Fisher 信息量 $I(\theta)$：



$$I(\theta) = -E\left[ \frac{\partial^2 \ln f(X; \theta)}{\partial \theta^2} \right] \quad \text{或} \quad Var\left( \frac{\partial \ln f(X; \theta)}{\partial \theta} \right)$$

计算 $n$ 个样本下的下界：



$$CRLB = \frac{[\tau'(\theta)]^2}{n I(\theta)}$$

计算你得到的 UMVUE 的方差 $Var(\hat{\theta})$。

**判定**：

- 若 $Var(\hat{\theta}) = CRLB$，则该估计量是**有效估计 (Efficient Estimator)**。
- 若 $Var(\hat{\theta}) > CRLB$，则它只是 UMVUE，**不是**有效估计。

### 3.1假设检验-似然比检验

### （1）求似然比检验

1. 建立假设 (Hypothesis Setting)

明确参数空间 $\Theta$ 的划分。

- **原假设** $H_0: \theta \in \Theta_0$。
- **备择假设** $H_1: \theta \in \Theta_1$。
- 注：通常 $\Theta_1 = \Theta \setminus \Theta_0$。

2. 构造似然比统计量 (Constructing the Statistic)

写出似然函数 $L(\theta)$，并构造统计量 $\lambda(x)$：



$$\lambda(x) = \frac{\sup_{\theta \in \Theta} L(\theta)}{\sup_{\theta \in \Theta_0} L(\theta)}$$

- **分子**：全空间 $\Theta$ 下的似然函数极大值（通常在全局 MLE $\hat{\theta}$ 处取得）。
- **分母**：受限空间 $\Theta_0$ 下的似然函数极大值（在限制 MLE $\hat{\theta}_0$ 处取得）。

3. 求解极大似然估计 (MLE Derivation)

分别计算 $\hat{\theta}$ 和 $\hat{\theta}_0$：

- 若全局 MLE $\hat{\theta}$ 落在 $\Theta_0$ 内，则 $\hat{\theta}_0 = \hat{\theta}$，此时 $\lambda(x) = 1$。
- 若全局 MLE $\hat{\theta}$ 落在 $\Theta_1$ 内，利用似然函数的单调性在边界处取得 $\hat{\theta}_0$。

4. 确定拒绝域 (Determining the Rejection Region)

似然比原理认为，当 $\lambda(x)$ 过大时（即观测结果与原假设偏离过远），应拒绝 $H_0$。

- 通过代入 MLE，将 $\lambda(x)$ 简化为关于某个常用检验统计量（如 $Z, T, \chi^2$）的函数。
- 依据 $\lambda(x) \geq c$ 导出具体的拒绝域形式（如 $Z \geq c_1$）。

5. 临界值与显著性水平 (Critical Value & Significance)

利用原假设下的分布确定常数 $c$。

- 通过 $P_{\theta_0} \{ \text{拒绝 } H_0 \} = \alpha$ 计算出临界值（如 $z_{1-\alpha}$）。
- 最终给出完整的拒绝域 $W$。

### （2）势函数

第一步：明确拒绝域 $W$

在进行任何势函数计算之前，必须先根据似然比检验（LRT）或其他准则确定拒绝域的具体形式。

- 通常拒绝域表现为检验统计量（如 $Z, T, \chi^2, F$）的函数。
- **示例**：$W = \{ z \geq z_{1-\alpha} \}$，其中 $z = \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}}$。

第二步：建立势函数定义式

势函数 $g(\theta)$ 的物理意义是：当参数真实值为 $\theta$ 时，样本落入拒绝域的概率。

- **数学表达式**：$g(\theta) = P_\theta(X \in W)$。
- 注意：此处的概率是在**真实分布**（参数为 $\theta$）下计算的，而非原假设分布。

第三步：标准化处理与分布代入

由于拒绝域通常是基于 $H_0$ 的边界值定义的，我们需要通过变换，将其转化为标准分布形式。

- 将拒绝域中的统计量改写为包含真实参数 $\theta$ 的标准随机变量（如标准正态变量 $N(0,1)$）。

- 计算技巧：

  

  $$g(\mu) = P_\mu \left( \frac{\bar{x} - \mu}{\sigma/\sqrt{n}} \geq z_{1-\alpha} + \frac{\mu_0 - \mu}{\sigma/\sqrt{n}} \right)$$

- 最终得到 $g(\theta)$ 关于累积分布函数（如 $\Phi(x)$）的解析式。

### （3）无偏检验

**分析势函数的单调性**

为了判别无偏性，需要确定 $g(\theta)$ 在整个参数空间 $\Theta$ 上的走势。

- **求导法**：对 $g(\theta)$ 求导，判断其在原假设空间 $\Theta_0$ 和备择假设空间 $\Theta_1$ 的增减性。
- **直观判定**：对于单侧检验，势函数通常是单调的；对于双侧检验，势函数通常在 $\theta_0$ 处取得极小值。

执行无偏性判定准则

**根据定义进行比对：**

1. **准则**：若对于任意 $\theta_1 \in \Theta_1$ 和任意 $\theta_0 \in \Theta_0$，均满足 $g(\theta_1) \geq g(\theta_0)$，则该检验为无偏检验。
2. **简便判定**：在显著性水平 $\alpha$ 下，只需验证：
   - 在 $H_0$ 范围内：$\sup_{\theta \in \Theta_0} g(\theta) = \alpha$（或 $\leq \alpha$）。
   - 在 $H_1$ 范围内：$\inf_{\theta \in \Theta_1} g(\theta) \geq \alpha$。

### 3.2假设检验，UMPT

### （1）UMPT

1.指数分布族

联合概率密度函数：形式$c(\theta)h(x_1,x_2,……,x_n)e^{w(\theta)T(x_1,x_2,……,x_n)}$,

判断$w(\theta)$严格单调。$T(x_1,x_2,……,x_n)$是完全充分统计量

**2.根据原假设和备择假设的方向，直接给出拒绝域 $W$ 的形式：**

- **右侧检验**：$H_0: \theta \leq \theta_0$ vs $H_1: \theta > \theta_0$
  - 拒绝域：$W = \{ T(x) > c \}$ 或 $\{ T(x) > c \text{ 且以概率 } \gamma \text{ 拒绝 } \}$（离散情况）。
- **左侧检验**：$H_0: \theta \geq \theta_0$ vs $H_1: \theta < \theta_0$
  - 拒绝域：$W = \{ T(x) < c \}$。

3.**利用显著性水平 $\alpha$ 解出临界值 $c$**

在边界点 $\theta = \theta_0$ 处，令犯第一类错误的概率等于 $\alpha$：



$$P_{\theta_0} \{ T(X) > c \} = \alpha$$

- **连续型**：直接通过 $T(X)$ 的分布（如 $\chi^2, N, \Gamma$ 分布）查表或计算出 $c$。
- **离散型**：可能需要寻找临界点 $c$ 和随机化概率 $\gamma$，使得 $P_{\theta_0}\{T>c\} + \gamma P_{\theta_0}\{T=c\} = \alpha$

## 第一章、参数估计

### 极大似然估计（MLE）

│     ├─ 原理：样本发生概率最大时的参数为估计值
│     ├─ 求解步骤
│     │  1. 构造似然函数：若总体密度/分布律为p(x;θ)，则L(θ)=∏p(x_i;θ)（样本独立同分布）
│     │  2. 取对数似然函数：lnL(θ)=∑lnp(x_i;θ)（简化求导计算）
│     │  3. 求导找极值点：对θ的各分量求偏导，令∂lnL(θ)/∂θ_j=0，解似然方程组
│     │  4. 验证极值：通过二阶导数或实际意义判断是否为最大值点（如均匀分布直接分析边界）（判断极大值能否取到）
│     │  5. 得到MLE：若似然方程组有解则为估计值，无解则结合分布特性确定（如顺序统计量）
│     ├─ 特点：充分利用分布信息，是充分统计量的函数
│     └─ 实例：二项分布、正态分布、均匀分布、Poisson分布参数估计

### 均方误差（MSE）准则

│  │  ├─ 定义：MSE=方差+偏差²
│  │  ├─ 计算步骤（★）
│  │  │  1. 计算估计量的期望E(T)=q(θ)+b(θ)（b(θ)为偏差）
│  │  │  2. 计算估计量的方差Var(T)=E(T²)-[E(T)]²
│  │  │  3. 代入MSE公式：MSE=Var(T)+b²(θ)

### 无偏性

│  │  ├─ 定义：估计量期望=真实参数（偏差=0）
│  │  ├─ 验证步骤
│  │  │  1. 写出估计量T(X₁,…,Xₙ)的表达式
│  │  │  2. 计算期望E(T)，化简后判断是否等于待估参数q(θ)
│  │  ├─ 特点：可能不存在或不唯一，无偏未必最优
│  │  └─ 实例：样本均值是总体均值的无偏估计，样本方差是总体方差的无偏估计

### $\star$一致最小方差无偏估计（UMVUE）

│  │  ├─ 定义：无偏估计类中方差最小
│  │  ├─ 求解步骤（★基于Lehmann-Scheffe定理）
│  │  │  1.寻找 完全充分统计量S(X) （通过因式分解定理或指数分布族性质）
│  │  │  2.构造待估参数q(θ)的一个 无偏估计φ(X)
│  │  │  3.计算条件期望T(X)=E(φ(X)|S(X))(一般用概率算)，得到UMVUE（若φ(X)已是S(X)的函数，则φ(X)即为UMVUE）
│  │  ├─ 存在性与唯一性：满足特定条件时存在且唯一
│  │  └─ 实例：正态总体均值（样本均值）、方差（样本方差）的UMVUE

联合概率密度函数：形式$c(\theta)h(x_1,x_2,……,x_n)e^{w(\theta)T(x_1,x_2,……,x_n)}$,

判断$w(\theta)$严格单调。$T(x_1,x_2,……,x_n)$是完全充分统计量

### $\star$信息不等式（Cramer-Rao不等式）

│  │  ├─ 核心：无偏估计的方差存在下界（C-R下界）
│  │  ├─ 计算步骤（★）
│  │  │  1. 计算Fisher信息量I(θ)=E[(∂lnp(X;θ)/∂θ)²]（或-I(θ)=E[∂²lnp(X;θ)/∂θ²]）
│  │  │  2. 样本Fisher信息量Iₙ(θ)=nI₁(θ)（I₁为单样本信息量）
│  │  │  3. 计算C-R下界：1/Iₙ(θ)，[q’(θ)]²/Iₙ(θ)
│  │  │  4. 判断有效性：若估计量方差等于下界则为有效估计
│  │  ├─ 相关概念：Fisher信息量（衡量参数信息含量）
│  │  └─ 特殊估计：有效估计（方差达到下界）、渐进有效估计（大样本下有效率趋近1）

统计量T所包含关于参数$θ$的Fisher信息量，用T的概率密度函数来计算

## 第二章 假设检验

### 一、基础概念

 核心定义
 原假设（H₀）与备择假设（H₁）：简单假设vs复合假设
 检验统计量：构造依据（如样本均值、似然比）
 拒绝域（W）与接受域（W^c）：样本空间的划分
 检验函数：非随机化检验（0-1取值）vs随机化检验（[0,1]取值）
 显著性水平（α）：控制第一类错误的上限
 两类错误与势函数
	第一类错误：H₀为真时拒绝H₀，概率α(θ)（去真）
	第二类错误：H₀为假时接受H₀，概率β(θ)（存伪）
	势函数（g(θ)）：H₀不成立时拒绝H₀的概率，g(θ)=1-β(θ)
	核心原理：Neyman-Pearson检验原理（控制第一类错误，最大化检验势）

### 二、假设检验通用步骤（★简答题高频考点）

1. 提出假设：明确H₀（θ∈Θ₀）与H₁（θ∈Θ₁）
2. 选取检验统计量：基于H₀的分布特性确定（如Z统计量、T统计量）
3. 给定显著性水平α（常用0.01、0.05、0.1）
4. 确定拒绝域：根据统计量分布和α计算临界值，划分W与W^c
5. 计算统计量值：代入样本观测值求解
6. 作出推断：统计量∈W则拒绝H₀，否则不拒绝H₀

### 三、检验方法

#### 似然比检验（★步骤类简答题）

│  │  ├─ 核心思想：比较H₁与H₀下的似然函数比值，比值过大则拒绝H₀
│  │  ├─ 通用步骤（★）
│  │  │  1. 分别在Θ（全（或者补）参数空间）和Θ₀（H₀参数空间）中求θ的极大似然估计（Ĥ、Ĥ₀）
│  │  │  2. 构造似然比统计量λ(x)=p(x,Ĥ)/p(x,Ĥ₀)（λ(x)≥1）
│  │  │  3. 化简λ(x)为h(T(x))，要求h是T(x)的单调函数，且H₀成立时T(x)分布已知
│  │  │  4. 确定临界值：**单调**增则由P(T≥c₁|H₀)=α求c₁，单调减则由P(T≤c₁|H₀)=α求c₁
│  │  │  5. 确定拒绝域：单调增对应W={T≥c₁}，单调减对应W={T≤c₁}
│  │  └─ 实例：正态总体（方差已知/未知）、离散总体的似然比检验

#### Neyman-Pearson引理（MPT构造依据）

│  │  ├─ 适用场景：简单H₀（θ=θ₀）vs简单H₁（θ=θ₁）
│  │  ├─ 核心结论：最优势检验（MPT）的拒绝域由似然比L(x)=p(x,θ₁)/p(x,θ₀)确定
│  │  └─ MPT检验函数（★步骤类）
│  │     1. 连续型L(x)：φ(x)=1（L(x)>k）、0（L(x)<k），k由P(L(x)≥k|H₀)=α确定
│  │     2. 离散型L(x)：φ(x)=1（L(x)>k）、r（L(x)=k）、0（L(x)<k），r=(α-P(L(x)>k|H₀))/P(L(x)=k|H₀)
（就是通过让期望=α）

#### 一致最优势检验（UMPT）

│  │  ├─ 定义：对所有θ∈Θ₁，势函数均不小于同水平其他检验
│  │  ├─ 存在条件：样本联合密度满足**指数分布族**形式p(x,θ)=d(θ)h(x)exp{c(θ)T(x)}，c(θ)严格单调
│  │  ├─ 构造步骤（★单边检验，θ≤θ₀vsθ>θ₀）
│  │  │  1. 验证样本密度符合指数分布族，确定T(x)和c(θ)的单调性（一般c(θ)增）
│  │  │  2. 构造检验函数：φ(x)=1（（拒绝区域））、r（T(x)=c（离散才考虑））、0（T(x)<c）
│  │  │  3. 确定c和r：由E_{θ₀}(φ*(x))=α求解
│  │  └─ 实例：指数分布、正态分布、Poisson分布的UMPT构造

## 第三章 贝叶斯分析

###  一、贝叶斯公式（核心工具）

连续型：π(θ|x)=[p(x|θ)π(θ)]/∫p(x|θ)π(θ)dθ，其中分母m(x)为x的边缘密度（无θ信息）
离散型：π(θ_i|x)=[p(x|θ_i)π(θ_i)]/∑p(x|θ_j)π(θ_j)

### 二、核心方法（★步骤简答题高频考点）

│  ├─ 1. 后验分布计算（★核心步骤题）
│  │  ├─ 适用场景：已知先验分布和样本，求参数θ的后验分布
│  │  ├─ 求解步骤（★）
│  │  │  1. 确定似然函数p(x|θ)：根据总体分布，写出样本联合密度（独立同分布时为乘积形式）
│  │  │  2. 指定先验分布π(θ)：基于先验信息（如无信息时用均匀分布、贝塔分布等共轭先验）
│  │  │  3. 计算联合分布h(x,θ)=p(x|θ)π(θ)：融合三种信息
│  │  │  4. 计算边缘密度m(x)=∫h(x,θ)dθ（连续型）或∑h(x,θ_i)（离散型）：归一化因子
│  │  │  5. 代入贝叶斯公式：得到后验分布π(θ|x)=h(x,θ)/m(x)
│  │  └─ 实例：二项分布+均匀先验→贝塔后验、离散参数（θ₁/θ₂）的后验概率更新
│  ├─ 2. 贝叶斯估计（★步骤题）
│  │  ├─ 原理：最小化期望风险，常用均方误差损失函数λ(θ,â)=(θ-â)²
│  │  ├─ 求解步骤（★）
│  │  │  1. 按“后验分布计算步骤”得到π(θ|D)（D为样本集）
│  │  │  2. 若采用均方误差损失，贝叶斯估计量为后验期望：â=E(θ|D)=∫θπ(θ|D)dθ（连续型）
│  │  │  3. 代入后验分布表达式，计算积分或求和，化简得到估计结果

## 第四章 多元统计基础

### 一、核心概念

│  ├─ 1. 随机向量与样本资料矩阵
│  │  ├─ 随机向量：p个随机变量的有序组合X=(X₁,X₂,…,Xₚ)ᵀ，分离散型和连续型
│  │  ├─ 样本资料矩阵：n行（样本数）×p列（变量数），每行是随机向量的一次观测
│  │  └─ 核心关联：样本资料矩阵是随机向量的观测实现，列数对应变量数p
│  ├─ 2. 分布函数与密度函数
│  │  ├─ 多元分布函数F(x)：P(X₁≤x₁,…,Xₚ≤xₚ)，取值[0,1]，满足单调性、有界性
│  │  ├─ 多元密度函数f(x)：连续型随机向量专属，F(x)是f(x)的p重积分，f(x)≥0且全空间积分=1
│  │  └─ 关联关系：f(x)是F(x)的p阶混合偏导数（连续点处）
│  ├─ 3. 独立性与数字特征
│  │  ├─ 随机向量独立：联合分布=各边缘分布乘积（分布函数/密度函数均成立）
│  │  ├─ 均值向量：μ=E(X)=(E(X₁),…,E(Xₚ))ᵀ，线性性质E(AX+b)=Aμ+b
│  │  ├─ 协方差阵：Σ=D(X)=E[(X-μ)(X-μ)ᵀ]，对称半正定（多数正定），性质D(AX)=AΣAᵀ，对角线元素
│  │  ├─ 相关阵：R=(rᵢⱼ)，rᵢⱼ=cov(Xᵢ,Xⱼ)/(√D(Xᵢ)√D(Xⱼ))，标准化数据的协方差阵=原相关阵
│  │  └─ 协方差阵与相关性：cov(X,Y)=0→线性不相关，正态分布下不相关=独立

### 二、多元正态分布（核心分布）

│  ├─ 1. 定义（三种等价形式）
│  │  ├─ 密度函数定义：f(x)=(2π)⁻ᵖ/²|V|⁻¹/²exp{-½(X-μ)ᵀV⁻¹(X-μ)}，V>0
│  │  ├─ 特征函数定义：φ(t)=exp{itᵀμ - ½tᵀVt}，V≥0
│  │  └─ 线性组合定义：对任意非零a∈Rᵖ，aᵀX~N(aᵀμ,aᵀVa)
│  ├─ 2. 核心性质（★简答题高频）
│  │  ├─ 唯一性：由均值向量μ和协方差阵V唯一确定，记为X~Nₚ(μ,V)
│  │  ├─ **线性变换不变性：Y=AX+b~Nₘ(Aμ+b,AΣAᵀ)（A为m×p常数阵**）
│  │  ├─ **独立性等价：正态向量分块后，互不相关=相互独立**
│  │  ├─ 平方分布：V>0时，(X-μ)ᵀV⁻¹(X-μ)~χ²(p)
│  │  └─ 条件分布：分块X=(X₁ᵀ,X₂ᵀ)ᵀ，给定X₂时X₁~Nₖ(μ₁·₂,V₁₁·₂)，其中μ₁·₂=μ₁+V₁₂V₂₂⁻¹(X₂-μ₂)，V₁₁·₂=V₁₁-V₁₂V₂₂⁻¹V₂₁
│  ├─ 3. 条件分布与偏相关
│  │  ├─ 条件期望E(X₁|X₂)：X₁对X₂的多元回归函数，回归系数矩阵=V₁₂V₂₂⁻¹
│  │  ├─ 偏相关系数：扣除其他变量影响后，两变量的相关系数，rᵢⱼ·ₖ₊₁…ₚ=σᵢⱼ·ₖ₊₁…ₚ/(√σᵢᵢ·ₖ₊₁…ₚ√σⱼⱼ·ₖ₊₁…ₚ)

## 第五章、非参数统计基础

### 符号秩和检验

符号秩和检验（Wilcoxon配对检验，检验H₀：F(x)=G(x)）
│     ├─ 求解步骤（★）
│     │  1. 提出假设：H₀：配对数据无差异，H₁：有差异
│     │  2. 计算差值与绝对值：Zᵢ=Xᵢ-Yᵢ，忽略Zᵢ=0，对|Zᵢ|赋予秩Rᵢ
│     │  3. 构造统计量：W⁺=Σ（Zᵢ>0时的Rᵢ）（正差值的秩和）
│     │  4. 确定临界值/分布：n≤20时查符号秩和检验表得c,d（d=n(n+1)/2 - c）；n>20时，W⁺~N(n(n+1)/4, n(n+1)(2n+1)/24)，构造标准化统计量~N(0,1)
│     │  5. 推断：W⁺≤d或W⁺≥c则拒绝H₀；大样本时|标准化统计量|>u₁₋α/₂则拒绝（W⁺取值适中）

填空题求$W^+$

#### 例题

某实验室测试了10名受试者在服用某种药物前后的某种生理指标，数据如下：

- **服用前**：$12, 15, 11, 14, 18, 13, 16, 17, 12, 14$

- 服用后：$14, 13, 12, 16, 20, 15, 17, 19, 11, 15$

  问：药物对该指标是否有显著影响？（$\alpha = 0.05$）



#### 第一步：建立假设

- **原假设 $H_0$**：配对差值的总体中位数为 $0$（即药物无影响）。
- **备择假设 $H_1$**：配对差值的总体中位数不为 $0$（即药物有影响）。

#### 第二步：计算差值 $d_i$

计算每组配对数据的差值 $d_i = X_{after} - X_{before}$：

- 差值序列：$+2, -2, +1, +2, +2, +2, +1, +2, -1, +1$

#### 第三步：求差值的绝对值并排秩

1. **剔除差值为 0 的项**：本例中无 0，样本量 $n = 10$。
2. **对绝对值 $|d_i|$ 进行从小到大排秩**。
3. **处理相同秩次（结）**：若绝对值相同，取其秩次的平均值。



| 差值 $d_i$ | 绝对值 $|d_i|$ | 符号 | 秩次 (Rank) | 符号秩 (Signed Rank) |
| :--------: | :------------: | :--: | :---------: | :------------------: |
|    $+1$    |      $1$       | $+$  |     $2$     |         $+2$         |
|    $+1$    |      $1$       | $+$  |     $2$     |         $+2$         |
|    $-1$    |      $1$       | $-$  |     $2$     |         $-2$         |
|    $+2$    |      $2$       | $+$  |    $7.5$    |        $+7.5$        |
|    $-2$    |      $2$       | $-$  |    $7.5$    |        $-7.5$        |
|    $+2$    |      $2$       | $+$  |    $7.5$    |        $+7.5$        |
|    $+2$    |      $2$       | $+$  |    $7.5$    |        $+7.5$        |
|    $+2$    |      $2$       | $+$  |    $7.5$    |        $+7.5$        |
|    $+2$    |      $2$       | $+$  |    $7.5$    |        $+7.5$        |
|    $+2$    |      $2$       | $+$  |    $7.5$    |        $+7.5$        |

#### 第四步：计算统计量 $W$

1. 计算正号秩和 $W^+$：$2 + 2 + 7.5 + 7.5 + 7.5 + 7.5 + 7.5 + 7.5 = 49$
2. 计算负号秩和 $W^-$：$2 + 7.5 = 9.5$
3. **检验统计量 $W$** 通常取 $W^+$ 和 $W^-$ 中的较小者：$W = \min(49, 9.5) = 9.5$。

#### 第五步：确定临界值并做出判断

1. **查表法**（适用于小样本 $n \leq 25$）：
   - 查 Wilcoxon 符号秩检验表，当 $n=10, \alpha=0.05$（双侧）时，临界值 $W_{0.05} = 8$。
2. **判断准则**：若计算得到的 $W \leq$ 临界值，则拒绝 $H_0$。
   - 本例中 $9.5 > 8$，故**接受 $H_0$**。
3. **结论**：在 $0.05$ 显著性水平下，不能认为该药物对生理指标有显著影响。
