# 📚 研究生核心课程备考资料库

> **研究生阶段核心课程复习与备考个人总结**
>
> 本仓库汇总了研究生期间两门核心数学课程及一门思政课程的个人备考资料，包含课程笔记、历年真题、知识点梳理、考试复盘与经验总结。所有内容均为本人修读时的一手整理，完全贴合授课重点与考试命题风格。

## ✨ 项目简介

本项目旨在为正在修读相关课程的同学提供一个高效的复习参考。资料涵盖了从**理论推导**到**实战刷题**的全过程，特别强化了对“考试套路”和“重难点”的解析。

### 涵盖学科：

- **🧮 数理统计**：涵盖参数估计、假设检验等核心推导。
- **🧩 矩阵理论**：侧重线性空间、分解理论及矩阵计算。
- **📜 思政专题**：提供《新时代中国特色社会主义理论与实践》的心得总结。
